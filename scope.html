  Scope | दायरा | कार्यक्षेत्र:

Variables are only accessible inside the region they are in created. This is known as दायरा/कार्यक्षेत्र(scope). वेरिएबल्स(variables ) केवल उस क्षेत्र के भीतर ही पहुंच योग्य हैं जहां वे बनाए गए हैं। इसे कार्यक्षेत्र या दायरा(scope) के रूप में जाना जाता है।(कहा जाता है):

 Method Scope |विधियों का कार्यक्षेत्र /दायरा

Variables which are declared directly inside a method are available anywhere in the method after(following) the line of code in which they were declared: एक विधि के अंदर सीधे घोषित किए गए वेरिएबल , उस पंक्ति(रेखा) जिसके जिसमें वे घोषित किए गए उसी पंक्ति(रेखा) के बाद ,उस विधि में कहीं भी उपलब्ध हैं |

Example:

उदाहरण:
सार्वजनिक कक्षा Main { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] आ){ // अ यहाँ काम नहीं करेगा , अ can not be used here अंक अ =100; // अ यहाँ काम करेगा ,अ can be used here तंत्र.बाहर.प्रिंट(अ); } }

 Block Scope|खंड कार्यक्षेत्र /दायरा

A block of code refers to all of the code between curly braces {}.
Variables that are declared inside blocks of code are only accessible by the code between the curly braces, which follows the line in which the variable was declared: कोड का एक खंड {}(कर्ली ब्रेसिज़) के बीच के सभी रेखाओं को संदर्भित करता है।
कोड के खंड के अंदर घोषित वेरिएबल , केवल {}(कर्ली ब्रेसिज़) के बीच के कोड द्वारा इस्तेमाल किए जा सकते हैं, वो भी केवल उस रेखा के बाद जिसमे उस वेरीअबल को घोषित किया गया है:

Example:

उदाहरण:
सार्वजनिक कक्षा Main { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] आ){ // अ यहाँ काम नहीं करेगा , अ can not be used here {// यह एक खंड है // यहाँ आप "अ" का इस्तेमाल नहीं कर सकते // क्योंकि यहाँ अभी तक उसे घोषित नहीं किया गया है अंक अ =100; // "अ" यहाँ काम करेगा क्योंकी // उसे घोशीत किया जा चुका है // और वो इसी खंड मे है | तंत्र.बाहर.प्रिंट(अ); } //यहाँ आप "अ" का इस्तेमाल नहीं कर सकते //क्योंकि वो इस खंड मे नहीं है } }

Note :

A खंड(block) of code may exist on its own or it can belong to an अगर (if),जब(while) or केलिए(for) statement. In the case of केलिए(for) statements, variables declared in the statement itself are also available inside the खंड कार्यक्षेत्र(block's scope).

ध्यान दे:

संहिता/कोड का एक खंड (block) अपने आप में मौजूद हो सकता है या यह किसी अगर (if),जब(जबकि) या केलिए से संबंधित हो सकता है । केलिए(for) कथनों के मामले में, कथन में घोषित चर(variable) भी खंड कार्यक्षेत्र (ब्लॉक के दायरे) के अंदर भी उपलब्ध हैं।
