 Enums| इअंक | परिगणना

An इअंक (enum/enumeration) is a special "वर्ग" ("class") that represents a group of constants (unchangeable variables, like अंतिम (final) variables.

To create an इअंक, use the इअंक(enum) keyword (instead of वर्ग (class) or अंतरापृष्ठ(interface)), and separate the constants with a comma. एक इअंक(enum/enumeration) एक विशेष "वर्ग" ("class") है जो स्थिरांक (न बदलने वाले अंकों) के एक समूह का प्रतिनिधित्व करता है (अपरिवर्तनीय चर(unchangable variables), जैसे अंतिम चर (final variables)) ।

एक इअंक (enum) बनाने के लिए , इअंक(enum) शब्द/कीवर्ड का उपयोग करें , और अपरिवर्तनीय चरों (unchangable variables) को अल्पविराम , से अलग करें।

Example:

उदाहरण:
इअंक स्तर { आसान, सामान्य, कठिन }

we can access इअंक(enum/enumeration) constants with the dot (.): हम इअंक(enum/enumeration) को बिंदु/डॉट (.) से प्राप्त कर सकते हैं।

स्तर मेरा_प्रश्न_पत्र_स्तर =स्तर.कठिन; //यहाँ आप मेरा_प्रश्न_पत्र_स्तर की जगह //कुछ और भी ले सकते है जैसे प्रश्न_पत्र, प्रश्नपत्र ,मेरास्तर ....

Generally In programming Enum is short for "enumerations", which means "specifically listed".In here the same concept of "enumerations" is taken as इअंक आम तौर पर प्रोग्रामिंग में Enum मतलब "enumerations" या "गणना" है, जिसका अर्थ है "विशेष रूप से सूचीबद्ध"। यहाँ समान अवधारणा को इअंक के रूप में लिया गया है।



 Enum inside a Class | एक वर्ग के आंदर इअंक

we can also have an इअंक(enum) inside a वर्ग(class): वर्ग(class) के अंदर हम इअंक(enum) भी रख सकते हैं:

Example:

उदाहरण:
वर्ग मुख्य { इअंक स्तर { आसान, सामान्य, कठिन } सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ स्तर मेरा_प्रश्न_पत्र_स्तर =स्तर.कठिन; तंत्र.बाहर.प्रिंट(मेरा_प्रश्न_पत्र_स्तर); } } // "कठिन" : लिख देगा

We can use इअंक(enum) any where in a file , in or out of a वर्ग (class), but not in a विधि (methord) . इअंक(enum) should always be out of a विधि (methord) . हम इअंक(enum) का उपयोग किसी फ़ाइल में कहीं भी, वर्ग (class) के अंदर या बाहर कर सकते हैं, लेकिन किसी विधि (methord) में नहीं | इअंक(enum) हमेशा विधि (methord) के बाहर होना चाहिए |



 Enum in a Switch Statement| बदले कथन मे इअंक

इअंक(Enums) are often used in बदले(switch) statements to check for corresponding values: इअंक(Enums) अक्सर बदले(switch) कथन में संबंधित मानों की जांच के लिए उपयोग किया जाता है:

Example:

उदाहरण:
इअंक स्तर { आसान, सामान्य, कठिन } वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ स्तर मेरा_प्रश्न_पत्र_स्तर =स्तर.कठिन; बदले (मेरा_प्रश्न_पत्र_स्तर){ यदि आसान: तंत्र.बाहर.प्रिंट(" आसान स्तर "); भंग; यदि सामान्य : तंत्र.बाहर.प्रिंट(" सामान्य स्तर "); भंग; यदि कठिन : तंत्र.बाहर.प्रिंट(" कठिन स्तर "); भंग; } } }



 Loop Through an Enum| इअंक मे घूमे/लूप करें

The इअंक(enum) type has a मान()("values()") method, which returns an "त्रम" ("array") of all इअंक(enum) constants. This विधि (method) is useful when you want to loop through the constants of an इअंक(enum): इअंक(enum) प्रकार में मान()("values()") विधि है, जो सभी इअंक(enum) का एक "त्रम" ("array") लौटाती है । यह विधि तब उपयोगी होती है जब आप इअंक(enum) के मानो के माध्यम से लूप करना चाहते हैं:

Example:

उदाहरण:
इअंक स्तर { आसान, सामान्य, कठिन } वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ केलिए (स्तर मेरा_प्रश्न_पत्र_स्तर: स्तर.मान() ) { तंत्र.बाहर.प्रिंट( मेरा_प्रश्न_पत्र_स्तर+ " " ); } } }


Difference between इअंक(Enum) and वर्ग(Class) :

An इअंक(enum) can, just like a वर्ग(class), have गुण (attributes) and विधियाँ (methods). The only difference is that इअंक(enum) constants are सार्वजनिक (public) , स्थिर (static) and अंतिम (final) (unchangeable - cannot be overridden).

An इअंक(enum) cannot be used to create वस्तुए (objects), and it cannot विरसेगा (extend) other वर्ग (classes) (but it can लागू अंतरापृष्ठ (implement interfaces)).

Why And When To Use इअंक(Enum)? :

Use इअंक(Enum) when you have values that you know aren't going to change, like month days, days, colors, deck of cards, etc.

एक इअंक(Enum) और वर्ग (class) के बीच अंतर :

एक इअंक(enum) में, वर्ग(class) की तरह, गुण/विशेषताएँ(attributes) और विधियाँ (methords) हो सकते हैं। फर्क सिर्फ इतना है कि इअंक(enum), न बदलने वाले सार्वजनिक (public), स्थिर (static) और अंतिम (final) (या कहे अपरिवर्तनीय - जिन्हे ओवरराइड नहीं किया जा सकता) होते हैं।

एक इअंक(enum) का उपयोग वस्तुए (objects) बनाने के लिए नहीं किया जा सकता है, और यह अन्य वर्गों (classes) कों विरास (extend) नहीं सकता है (लेकिन यह अंतरपृष्ठ लागू कर सकता है (can implement interfaces))।

इअंक(Enum) क्यों और कब इस्तेमाल करें ? :

इअंक(Enum) का उपयोग तब करें जब आपके पास ऐसे मान(values) हैं जो आप जानते हैं कि वे बदल ने वाले नहीं हैं, जैसे महीने के दिन, दिन, रंग, ताश के पत्तों की गड्डी आदि।
