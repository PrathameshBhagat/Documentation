  Data Types| डेटा के प्रकार :

As we saw in the previous chapter, a variable must be a specified by a data type: जैसा कि पिछले अध्याय में हमने देखा की ,एक variable एक निर्दिष्ट डेटा प्रकार होना चाहिए:

अंक Num = 5; फ्लोट FloatNum = 5.99f; वर्ण Letter = 'D'; बूलियन Bool = सच; डोर Text = "Hello";
Data types are divided into two groups:
  • आदिम(पुराने/आसान) डेटा प्रकार(Primitive data types) -consist ofबाइट,छोटा,अंक,लंबा,फ्लोट,बड़ा,बूलियनand वर्ण
  • गैर-आदिम डेटा प्रकार(Non-primitive data types) -these include डोर, Arrays and कक्षा (we will learn more about these in a later chapter)
 डेटा प्रकारों को दो समूहों में विभाजित किया गया है:
  • आदिम(पुराने/आसान) डेटा प्रकार(primitive) -बाइट,छोटा,अंक,लंबा,फ्लोट,बड़ा,बूलियन और वर्ण शामिल हैं
  • गैर-आदिम डेटा प्रकार(non-primitive) - जैसे डोर,Arrays , और कक्षा(हम बाद के अध्याय में इनके बारे में अधिक जानेंगे)

 Primitive Data Types|आदिम(पुराने/आसान) डेटा प्रकार :

 Primitive Data Types|आदिम(पुराने/आसान) डेटा प्रकार:

Primitive number types are categorised into two groups:
Integer types stores whole numbers, positive or negative (E.g: 123 or -456), with no decimals. Valid types are बाइट,छोटा,अंक andलंबा. Which type you should use, depends on the numeric value.

Floating point types represents numbers with a fractional part, containing one or more decimals. There are two types: फ्लोट and बड़ा. आदिम संख्या प्रकारों को दो समूहों/गुटों में बाटा गया है:
पूर्णांक प्रकार दशमलव के बिना पूर्ण संख्याओं, धनात्मक या ऋणात्मक (जैसे 123 या -456) को संग्रहीत करते हैं। मान्य प्रकार बाइट,छोटा,अंक andलंबा होते हैं। आपको किस प्रकार का उपयोग करना चाहिए, संख्यात्मक मान पर निर्भर करता है।

फ़्लोटिंग पॉअंक प्रकार एक आंशिक(दशमलव) भाग के साथ संख्याओं का प्रतिनिधित्व करते हैं, जिसमें एक या अधिक दशमलव होते हैं। दो प्रकार के होते हैं: फ्लोट और बड़ा।

 बाइट

The बाइट data type can store whole numbers from -128 to 127.
It is used instead of अंक or other integer types to save memory and when you know that the value will be within -128 and 127: बाइट डेटा प्रकार संपूर्ण संख्याओं(whole no.s) को -128 से 127 तक संग्रहीत/धारण कर सकता है।
इसका उपयोग कंप्युटर की स्मृति को सहेजने केलिए अंक या अन्य पूर्णांक प्रकारों के बजाय किया जा सकता है जब आप निश्चित हों कि मान -128 और 127 के भीतर होगा:

बाइट num =100;//-128 to 127 तंत्र.बाहर.प्रिंट(num);

 छोटा

The छोटा data type store's whole numbers from -32768 to 32767: छोटा डेटा प्रकार पूर्ण संख्याओं को -32768 से 32767 तक संग्रहीत/धारण कर सकता है:

छोटा num =5000;//-32768 to 32767 तंत्र.बाहर.प्रिंट(num);

 अंक

The अंक data type store's whole numbers from -2147483648 to 2147483647. In general,the अंक data type is the preferred data type to create variables with a numeric value.: अंक डेटा प्रकार संपूर्ण संख्याएँ -2147483648 से 2147483647 तक धारण करता है . सामान्य तौर पर, अंक डेटा प्रकार संख्यात्मक मान के साथ variable बनाने केलिए पसंदीदा डेटा प्रकार है।:

अंक num =100000;//-2147483648 to 2147483647 तंत्र.बाहर.प्रिंट(num);

 लंबा

लंबा ,This data type store's whole numbers from -9223372036854775808 to 9223372036854775807.It is used when अंक is not large enough to store the value. Note that you should end the value with an "ल": लंबाइस डेटा प्रकार संपूर्ण संख्या -9223372036854775808 से 9223372036854775807 तक धारण करता है .इसका उपयोग तब की जाती है जब अंक मान(value) को संग्रहीत करने केलिए पर्याप्त बड़ा नहीं होता है। ध्यान दें कि आपको "ल" के साथ मान समाप्त करना होता है :

लंबा num =15000000000ल;//-9223372036854775808 to 9223372036854775807 तंत्र.बाहर.प्रिंट(num);



 Fractional types|आंशिक प्रकार

We should use a fractional type/floating point type whenever we need a number with a decimal, such as 93.99 or 3.1424 .The फ्लोट and बड़ा data types can store fractional numbers.
Note that you should end the value with an "ल" for फ्लोट and "ब" for बड़ा: जब भी हमें दशमलव के साथ किसी संख्या की आवश्यकता होती है, जैसे कि 93.99 या 3.1424, तो हमें एक आंशिक प्रकार /फ़्लोटिंग पॉअंक प्रकार का उपयोग करना चाहिए। फ्लोट और बड़ा डेटा प्रकार आंशिक/दशमलव संख्याओं को धारण कर सकते हैं।
ध्यान दें कि आपको फ्लोट केलिए "ल" और बड़ा केलिए "ब" के साथ मान समाप्त करना चाहिए:

फ्लोट :

फ्लोट num = 5.75फ; तंत्र.बाहर.प्रिंट(num);

बड़ा :

बड़ा num = 5.75ब; तंत्र.बाहर.प्रिंट(num);

Note :

फ्लोट or बड़ा?
The precision of a fraction indicates number of digits the value it can have after the decimal point. . The precision of फ्लोट is only six or seven decimal digits, while बड़ा variables have a precision of nearly 15 digits.So it is safer to use double for most calculations (but it uses lot of memory space as compared to others).

ध्यान दे:

फ्लोट या बड़ा?
एक भिन्न/आंशिक संख्या की परिशुद्धता दशमलव बिंदु के बाद होने वाले मान(value)के अंकों की संख्या को दर्शाती है।फ्लोट की सटीकता केवल 6 या 7 दशमलव अंक है, जबकि बड़ा में लगभग 15 अंकों की सटीकता है। इसलिए अधिकांश गणनाओं केलिए बड़ा का उपयोग करना सुरक्षित है (लेकिन यह दूसरों की तुलना में बहुत अधिक मेमोरी स्पेस का उपयोग करता है)


Scientific Numbers|वैज्ञानिकी संख्या

A fractional number can also be a scientific number with an "e" or "इ" to indicate the power of 10: एक आंशिक संख्या 10 की शक्ति(power of 10/exponent of 10) को दर्शित करने केलिए "e" या "इ" के साथ एक वैज्ञानिक संख्या भी हो सकती है

फ्लोट num = 12E4फ; तंत्र.बाहर.प्रिंट(num); बड़ा num1 = 35e3ब; तंत्र.बाहर.प्रिंट(num1);

Boolean Types|दोहरा प्रकार

Many times in programming, we will need a data type that can only have one of two values, such as:
  • YES / NO
  • TRUE / FALSE
For this we have a दोहरा data type, which only take's one of the values true or false: प्रोग्रामिंग में कई बार, हमें एक डेटा प्रकार की आवश्यकता होगी जिसमें केवल दो मानों(values) में से एक हो सकता है, जैसे:
  • हाँ /नहीं
  • बंद/शुरू

इसकेलिए हमारे पास एक दोहरा डेटा प्रकार है, जो केवल सच या झूठ मानों में से एक लेता है:
बूलियन learningisfun = सच; तंत्र.बाहर.प्रिंट(learningisfun); बूलियन learningisnotfun = झूठ; तंत्र.बाहर.प्रिंट(learningisfun);

Character Data Type |वर्ण प्रकार

The वर्ण data type is used to store a single character or a letter. The वर्ण data type must be surrounded by single quotes' ', like 'A' or 'c' वर्ण डेटा प्रकार का उपयोग किसी एकल वर्ण या अक्षर को संग्रहीत(धारण) करने केलिए किया जाता है। चार डेटा प्रकार को एकल उद्धरणों' ' से घिरा होना चाहिए, जैसे 'अ' या 'ब'

वर्ण c='a'; तंत्र.बाहर.प्रिंट(c);// prints 'a'

String | डोर

The डोरdata type store's a sequence of characters(text)or वर्ण elements. डोरdata type values must be surrounded by double quotes" ": डोरडेटा प्रकार वर्णों का संग्रह है या कहे वर्ण तत्वों का एक अनुक्रम है। डोरडेटा प्रकार मान(values) दोहरे उद्धरणों" " से घिरे होने चाहिए:

डोर a ="Hello"; तंत्र.बाहर.प्रिंट(a);

Note :

A डोर type is actually a non-primitive data type, because it refers to an object. The डोर object has methods that are used to perform certain operations on it. Don't worry if you don't understand the term "object" just yet. We will learn more about डोर and objects in a later chapter.

ध्यान दे:

एक डोर प्रकार वास्तव में एक गैर-आदिम डेटा प्रकार है, क्योंकि यह किसी ऑब्जेक्ट को संदर्भित करता है। डोर ऑब्जेक्ट में ऐसी विधियाँ हैं जिनका उपयोग उस पर कुछ काम करने केलिए किया जाता है। चिंता न करें यदि आप अभी तक "ऑब्जेक्ट" शब्द को नहीं समझते हैं। हम बाद के अध्याय में डोर और वस्तुओं के बारे में अधिक जानेंगे।

Non-Primitive Data Type|आदिम(पुराने/आसान) डेटा प्रकार

Non-primitive data types are also called reference types as they refer to "objects".
The main difference between a primitive and a non-primitive data type is as follows:
  • Primitive types are predefined (already defined). Non-primitive types are created by the programmer and not defined already (except for डोर).
  • Non-primitive types can be used to call methods to perform certain operations, while primitive types cannot.
  • A primitive type always has some value, but non-primitive types can be null.
  • The size of a primitive type depends on the data type,incase of non-primitive types they all have the same size.
  • Examples of non-primitive types are Strings, त्रम(Arrays),कक्षा (classes),अंतरापृष्ठ (Interface), etc.
We will learn more about these in a later/respective chapters . गैर-आदिम(पुराने) डेटा प्रकार"ऑब्जेक्ट्स" को संदर्भित करते हैं।
एक आदिम और एक गैर-आदिम डेटा प्रकार के बीच मुख्य अंतर निम्नानुसार है:
  • आदिम प्रकार पूर्वनिर्धारित हैं (पहले से परिभाषित)। गैर-आदिम प्रकार प्रोग्रामर द्वारा बनाए जाते हैं जो पहले से परिभाषित नहीं होते हैं (डोर को छोड़कर)।
  • गैर-आदिम प्रकारों का उपयोग कुछ संचालन या काम करने केलिए उनकी विधियों को बुलाया जा सकता है, जबकि आदिम प्रकार नहीं कर सकते हैं।
  • एक आदिम प्रकार का हमेशा कुछ मूल्य(वैल्यू) होता है, लेकिन गैर-आदिम प्रकार शून्य हो सकते हैं।
  • एक आदिम प्रकार का आकार उसके प्रकार पर निर्भर करता है,गैर-आदिम प्रकारों के मामले में उन सभी का आकार समान होता है।
  • गैर-आदिम प्रकारों के उदाहरण डोर(string), त्रम(Arrays), कक्षा (classes), अंतरापृष्ठ (Interface),आदि हैं।
हम बाद के अध्याय में इनके बारे में अधिक जानेंगे।
