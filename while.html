  Loops|लूपस |घुमावदार वक्र:

Loops are ones which can execute a block of code as long as a specified or given condition is reached.
These are very handy because they save time,help reduce errors, and they make code much more readable. घुमावदार वक्र/लूप वे हैं जो कोड के एक ब्लॉक को निष्पादित(execute) कर सकते हैं जब तक कि एक निर्दिष्ट या दी गई स्थिति तक पहुंच न जाए।
ये बहुत आसान होते हैं क्योंकि वे समय बचाते हैं, त्रुटियों को कम करने में मदद करते हैं, और वे कोड को अधिक पठनीय बनाते हैं।

 While Loop|जब-तक कथन

This loops through a block of code as long as a specified/given condition is true(outputs true): यह कोड के एक ब्लॉक के माध्यम से लूप करता है(बार बार निष्पादित/execute करता है) जब तक कि निर्दिष्ट/दी गई स्थिति सत्य है(सत्य साबित होती है):

Syntax:

वाक्य विन्यास(syntax):
जब-तक(/*condition*/){ //code to be executed if the given condition is true }


In the following example , the given code in the loop will run, over and over again, as far as a given variable (i) is less than 5: निम्नलिखित उदाहरण में, लूप के आंदर दिया गया कोड बार-बार चलेगा, जहां तक दिया गया चर (i) 5 से कम है:

Example:

उदाहरण:


अंक i = 0; जब-तक(i < 5){ तंत्र.बाहर.प्रिंट(i); i++; }

Note :

!!Do not forget to increase the variable in the condition, otherwise the loop will never end and will continue consuming resourses infinitely if not terminated by an external program !!

ध्यान दे:

!! variable को बढ़ाने मे चूक ना करे , अन्यथा लूप कभी खत्म नहीं होगा और सिस्टम के उपकरणों का उपभोग करना जारी रखेगा !!


 The करो..जब-तक(do/while) Loop|करो..जब-तक कथन

The करो..जब-तक loop is a variant of the जब-तकloop.It loop will execute the given code block once, before checking if the condition is true or computes to be true, after this it will repeat the loop for as far as the condition is true. करो..जब-तक लूप जब-तकलूप का एक संस्करण है। यह लूप दिए गए कोड ब्लॉक को एक बार निष्पादित(execute) करेगा, यह जांचने से पहले कि क्या स्थिति सही है या सत्य होने की गणना करता है, इसके बाद यह लूप को दोहराएगा जहां तक स्थिति सच है।

Syntax:

वाक्य विन्यास(syntax):
करो{ //code to be executed if the given condition is true } जब-तक(/*condition*/)

Example:

This example uses a करो..जब-तक loop. The loop will always get executed at least once, even if the condition is computed to be false, as the code block is executed before the given condition is tested:

उदाहरण:

यह उदाहरण एक करो.. जब-तक का उपयोग करता है ।यह लूप हमेशा कम से कम एक बार निष्पादित(execute) किया जाएगा, भले ही स्थिति की गणना झूठी/गलत होने केलिए की जाती है, क्योंकि दिए गए स्थिति का परीक्षण करने से पहले कोड ब्लॉक निष्पादित(execute) किया जाता है:
करो{ तंत्र.बाहर.प्रिंट(i); i++; } जब-तक(i < 5)

Note :


!!Do not forget to increase the variable in the condition, otherwise the loop will never end and will continue consuming resourses infinitely if not terminated by an external program !!.

Also note that we can use जब in place of जब-तक and करो..जबin place of करो..जब-तक to make the code readable and short both are just for convinense and are exactly the same .Always prefer जब as जब-तक would soon be removed in future release .
