 Modifiers|संशोधक:

By now, you are quite familiar with the सार्वजनिक keyword that appears in almost all of our examples: अब तक, आप सार्वजनिक से परिचित हो चुके हैं जो हमारे लगभग सभी उदाहरणों में दिखाई देता है:

सार्वजनिक स्थिर रिक्त जरूरी(डोर [] क ){}

The सार्वजनिक keyword is an पहुँच संशोधक(access modifier), meaning that it is used to set the access level for classes, attributes, methods and constructors.
We divide संशोधक(modifiers) into two groups:
  • पहुँच संशोधक(Access Modifiers) - controls the access level
  • गैर पहुँच संशोधक(Non-Access Modifiers) - do not control access level, but provides other functionality
 सार्वजनिक एक पहुँच संशोधक है, जिसका अर्थ है कि इसका उपयोग कक्षाओं/वर्गों (classes), विशेषताओं(attributes), विधियों(methords) और निर्माणकर्ताओं(constructors) के लिए पहुँच स्तर स्पष्ट करने के लिए किया जाता है।
हम संशोधक को दो समूहों में विभाजित करते हैं:
  • पहुँच संशोधक(Access Modifiers)- पहुँच स्तर(access level) को नियंत्रित करता है
  • गैर पहुँच संशोधक(Non-Access Modifiers) - पहुँच स्तर(access level) नियंत्रित नहीं करते हैं, लेकिन अन्य कार्यक्षमता प्रदान करते हैं

 Access Modifiers|पहुँच संशोधक

For कक्षा/वर्ग(classes), you can use either सार्वजनिक or आम(default): कक्षा/वर्ग(classes) के लिए, आप सार्वजनिक या आम(default) का उपयोग कर सकते हैं:

ModifierसंशोधकDescriptionवर्णन
सार्वजनिकThe वर्ग(class) is accessible by any other classवर्ग किसी अन्य वर्ग द्वारा पहुँचा जा सकता है
आमThe वर्ग(class) is only accessible by वर्ग(class)'s in the same package.When you don't specify the modifier this is assumed to be linked automatically. We will learn more about packages later.वर्ग(class) केवल उसी पैकेज में वर्गों(classes) द्वारा ही पहुँचा जा सकता है। जब आप संशोधक निर्दिष्ट नहीं करते हैं तो यह स्वचालित रूप से जुडा माना जाता है। हम बाद में पैकेजों के बारे में और जानेंगे।



 Public Classes|सार्वजनिक वर्ग/कक्षा :

We have seen some classes with access modifier सार्वजनिक and other classes without it(आमवर्ग (default classes) . The difference between the two is that सार्वजनिक वर्ग (public class) is accessible to all other classes belonging or not belonging to the same package but a आम(default) class cannot be accesseed outside of the package it belongs to it can only be accessed in the same package (we will learn about package later).

Also a सार्वजनिक वर्ग (public class) should always be placed in a file with same name as the class.That means a class(वर्ग) named कार should be in file named "कार.h4" .
Also a single file cannot have more than one सार्वजनिक वर्ग (public class) but it can have more than one आमवर्ग (default classes). हमने सार्वजनिक संशोधक के साथ कुछ कक्षाएं/वर्ग और इसके बिना अन्य कक्षाएं देखी हैं (आमवर्ग (default classes)। दोनों के बीच अंतर यह है कि सार्वजनिक वर्ग (public class) अन्य सभी वर्गों के लिए उपलब्ध है जो उसी पैकेज से संबंधित हो या ना हों लेकिन एक आम वर्ग (default class)को उस पैकेज के बाहर इस्तेमाल नहीं किया जा सकता है जो इससे संबंधित है इसे एक ही पैकेज में इस्तेमाल किया जा सकता है (पैकेज के बारे में हम बाद में सीखेंगे)। इसका मतलब है कि "कार" नामक एक वर्ग (class) "कार.h4" नाम की फ़ाइल में होना चाहिए।
साथ ही एक फ़ाइल में एक से अधिक सार्वजनिक वर्ग (public class) नहीं हो सकते ।
लेकिन एक फ़ाइल में एक से अधिक आमवर्ग (default classes) हो सकते हैं ।

For now just understand : The following code Should always lie in a file named "मुख्य.h4" as it has a सार्वजनिक वर्ग (public class) named "मुख्य" .(If not it would generate an error).

इसमें अन्य गैर सार्वजनिक वर्ग(non - public class) हो सकते हैं यानी इसमें कई अन्य आम वर्ग (default class) हो सकते हैं जैसे "कार" लेकिन इसमें एक भी सार्वजनिक वर्ग(public class) नहीं हो सकती {"मुख्य" को छोड़कर}।
