उपयोगकर्ता से निवेश/सहयोग ले :

The स्कैनर वर्ग (Scanner class) is used to get user input.

To use the स्कैनर वर्ग (Scanner class), create an object of the class and use any of the available methods found in the स्कैनर वर्ग (Scanner class). In our example, we will use the अगलीरेखा() विधि (nextLine() method), which is used to read डोर (Strings): स्कैनर वर्ग (Scanner class) का उपयोग उपयोगकर्ता इनपुट प्राप्त करने के लिए किया जाता है।

स्कैनर वर्ग (Scanner class) का उपयोग करने के लिए, वर्ग का एक ऑब्जेक्ट बनाएँ और स्कैनर वर्ग (Scanner class) में उपलब्ध किसी भी उपलब्ध विधि का उपयोग करें। हमारे उदाहरण में, हम अगलीरेखा() विधि (nextLine() विधि) का उपयोग करेंगे, जिसका उपयोग डोर (Strings) को लेने/पढ़ने के लिए किया जाता है:

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ स्कैनर मेरिवस्तू = नई स्कैनर (तंत्र.मे); डोर उपयोगकर्तानाम ; तंत्र.बाहर.प्रिंट("उपयोगकर्ता का नाम बताए "); उपयोगकर्तानाम = मेरिवस्तू.अगलीरेखा(); तंत्र.बाहर.प्रिंट("उपयोगकर्ता का नाम : " + उपयोगकर्तानाम); } }



 Input Types |इनपुट प्रकार :

In the example above, we used the अगलीरेखा() method, which is used to read डोर (Strings). To read other types, look at the table below: उपरोक्त उदाहरण में, हमने अगलीरेखा() विधि का उपयोग किया, जिसका उपयोग डोर (Strings) को प्राप्त करने(पढ़ने) के लिए किया जाता है। अन्य प्रकारों को प्राप्त करने(पढ़ने) के लिए, नीचे दी गई तालिका देखें:

MethordविधिDescriptionवर्णन/जानकारी
अगलादोहरा() Reads a दोहरा value from the user उपयोगकर्ता से दोहरा मान(वैल्यू) लेता/पढ़ता है
अगलाबाइट() Reads a बाइट value from the user उपयोगकर्ता से बाइट मान(वैल्यू) लेता/पढ़ता है
अगलाबड़ा() Reads a बड़ा value from the user उपयोगकर्ता से बड़ा मान(वैल्यू) लेता/पढ़ता है
अगलाफ्लोट() Reads a फ्लोट value from the user उपयोगकर्ता से फ्लोट मान(वैल्यू) लेता/पढ़ता है
अगलाअंक() Reads a अंक value from the user उपयोगकर्ता से अंक मान(वैल्यू) लेता/पढ़ता है
अगलालंबा() Reads a लंबा value from the user उपयोगकर्ता से लंबा मान(वैल्यू) लेता/पढ़ता है
अगलाछोटा() Reads a छोटा value from the user उपयोगकर्ता से छोटा मान(वैल्यू) लेता/पढ़ता है

In the example below, we use different विधियाँ(methods) to read data of various types: नीचे दिए गए उदाहरण में, हम विभिन्न प्रकार के मान या डेटा को पढ़ने के लिए विभिन्न विधियों(methods) का उपयोग करते हैं:


Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ स्कैनर मेरिवस्तू = नई स्कैनर (तंत्र.मे); डोर उपयोगकर्तानाम ; फ्लोट पगार ; अंक उम्र ; तंत्र.बाहर.प्रिंट("उपयोगकर्ता का नाम बताए "); उपयोगकर्तानाम = मेरिवस्तू.अगलीरेखा(); तंत्र.बाहर.प्रिंट("उपयोगकर्ता की उम्र ,पगार बताए "); पगार = मेरिवस्तू.अगलाफ्लोट(); उम्र = मेरिवस्तू.अगलाअंक(); तंत्र.बाहर.प्रिंट("उपयोगकर्ता का नाम : " + उपयोगकर्तानाम +" उम्र "+उम्र+ " पगार "+पगार); } }



Difference between अगलीरेखा and अगलाअंक , अगलाफ्लोट ....

All other विधियाँ(methords) have respective डाटा प्रकार (data type) in विधि का नाम (methord name) अंक => अगलाअंक() ,फ्लोट => अगलाअंक() .. . in them but अगलीरेखा does not have it. Also it has a change in its letter from others ,it has ी(अगल ी रेखा) in place of ा(अगल ा अंक).

Note :

If you enter wrong input (e.g. text in a numerical input), you will get an त्रुटि संदेश (exception/error message) ( We will learn about them in next topic itself)


Also Note that the online compiler can have issues in taking uesr input , it is also possible that programs asking user input may not work at all.

अगलीरेखा और अगलाअंक , अगलाफ्लोट ... . के बीच अंतर,

अन्य सभी विधियाँ (विधि) में संबंधित डेटा प्रकार (data type) में विधि का नाम (methord name) अंक => अगलाअंक() ,फ्लोट => अगलाफ्लोट() .. आदि उनमें लेकिन अगलीरेखा मे नहीं है। साथ ही इसकी मात्रा में अन्य से परिवर्तन हुआ है, इसमें ा(अगल ा आंक) के स्थान पर ी(अगल ी लाइन) है।

ध्यान दे :

यदि आप गलत सामग्री/इनपुट प्रदान करते हैं (जैसे संख्यात्मक सामग्री/इनपुट में लेखन(text)), तो आपको एक त्रुटि संदेश (अपवाद/त्रुटि संदेश) मिलेगा (इनके बारे में हम अगले विषय में ही जानेंगे)।


यह भी ध्यान दे online compilers को उपयोगकर्ता से निवेश/सहयोग (user input) मे दिक्कत हो सकती है यह भी हो सकता है की वो काम ही ना करे |
