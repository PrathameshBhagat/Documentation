 Arrays|त्रम|सरणियाँ|व्यूह(रचना):

Arrays store multiple values in a single variable, instead of declaring manny separate variables for each value.
To declare an array, define the variable type with square brackets[ ]: सरणी/त्रम प्रत्येक मान(value) केलिए बहुत से अलग चर(variable) घोषित करने के बजाय, एक एकल चर(variable) में कई मानों(values) को संग्रहीत करते हैं।
एक सरणी/त्रम घोषित करने केलिए, वर्ग कोष्ठक[] के साथ चर(variables) प्रकार को परिभाषित करें[]:

डोर[] car ;

We have now declared a variable which holds an array of डोर(strings). To insert values in it, we can place the values in a comma-separated list, inside curly braces(brackets){}: अब हमने एक चर(variable) घोषित किया है जिसमें डोर (स्ट्रिंग्स) की एक सरणी(त्रम) है। इसमें मान(value) प्रदान करने केलिए, हम मूल्यों(values) को एक अल्पविराम-पृथक सूची में, घुंघराले/घुमावदार ब्रेसेस(ब्रैकेट) {} के अंदर रख सकते हैं:

डोर[] bikes = {"ktm", "Kavasaki-ninja ", "susuki", "Honda"};

डोर[] bikes = {100,200,300,400,500 };

 Access the Elements of an Array|एक सरणी/त्रम के तत्वों तक पहुँचें:

We can access an array's elements by referring to their index numbers .
The following statement accesses the value of the first element in the cars array : हम उनकी अनुक्रमणिका(अनुक्रमांक) संख्याओं का उल्लेख करके सरणी के तत्वों तक पहुँच सकते हैं।
निम्नलिखित कथन कार सरणी में पहले तत्व के मूल्य तक पहुँचता है:

Example:

उदाहरण:


डोर [] bikes ={"Ktm", "Kavasaki-Ninja ", "susuki", "Honda"}; तंत्र.बाहर.प्रिंट(bikes[0] ); // Outputs Ktm

Note :

All Array indexes start with 0: [0] is the first element. [1] is the second element,and so on .

ध्यान दे:

सभी सरणी/त्रम इंडेक्स 0 से शुरू होते हैं: [0] पहला तत्व है। [1] दूसरा तत्व है, और इसी तरह आगे ।

 Loop Through an Array|एक सरणी/त्रम से लूप

We can loop through the array elements with the केलिएloop, and use the लंबाई property to specify how many times the loop should run.
The following example outputs all elements in the bikes array: हम केलिए लूप के साथ सरणी/त्रम तत्वों के माध्यम से लूप कर सकतेहैं, और यह निर्दिष्ट करने केलिए लंबाई गुण का उपयोग कर सकते हैं कि लूप को कितनी बार चलना चाहिए।
निम्न उदाहरण bikes सरणी/त्रम में सभी तत्वों को आउटपुट करता है:

Example:

उदाहरण:


डोर [] bikes ={"Ktm", "Kavasaki-Ninja ", "susuki", "Honda"}; केलिए(अंक i = 0; i < bikes.लंबाई ; i++) { तंत्र.बाहर.प्रिंट(bikes[i]); }


 Loop Through an Array with For-Each|प्रत्येककेलिए एक सरणी/त्रम से लूप :

We also have a प्रत्येककेलिए("for-each") loop, which is used exclusively in order to loop through the elements in an array: हमारे पास एक प्रत्येककेलिए ("for-each") लूप भी है, जिसका उपयोग विशेष रूप से एक सरणी/त्रम में तत्वों से लूप करने केलिए किया जाता है:

Syntax:

वाक्यविन्यास:


केलिए (प्रकार variable : arrayname) { ... }

Example:

This example outputs all elements in the bikes array, using a प्रत्येककेलिए ("for-each") loop:

उधारण:

यह उदाहरण bikes सरणी/त्रम में सभी तत्वों को आउटपुट करता है, एक प्रत्येककेलिए ("for-each") लूप का उपयोग करके
डोर [] bikes ={"Ktm", "Kavasaki-Ninja ", "susuki", "Honda"}; केलिए(अंक i : bikes) { तंत्र.बाहर.प्रिंट(i); }

The above example can be read as : for each डोर element (called i - as in index) in bikes, print out the value of i.
If we compare the केलिए(for loop) and प्रत्येककेलिए(for-each) loop, we will see that the प्रत्येककेलिए(for-each) method is easier to write/code, it does not require a counter (using the लंबाई property), and is more readable. उपरोक्त उदाहरण को इस प्रकार पढ़ा जा सकता है:bikes में प्रत्येक डोर तत्व (जिसे i - index में कहा जाता है) केलिए, i के मान(value) को प्रिंट करें।
यदि हम तुलना करते हैं, तो हम देखेंगे कि प्रत्येककेलिए विधि लिखना आसान है, इसकेलिए काउंटर (लंबाई संपत्ति का उपयोग करने) की आवश्यकता नहीं है, और ये अधिक पठनीय है।

 Multidimensional Arrays|बहुआयामी सरणी/त्रम :

Multidimensional array is actually an array of arrays.
So to create a two-dimensional array, add each array within its own set of curly braces{{},{},{}}: बहुआयामी सरणी/त्रम वास्तव में सरणियों की एक सरणी है।
तो एक द्वि-आयामी सरणी/त्रम बनाने केलिए, प्रत्येक सरणी/त्रम को घुंघराले/घुमावदार कोष्ठक(curly brackets) के अपने सेट के भीतर जोड़ें :{{},{},{}}

Example:

उधारण:


अंक [][] Numbers = { {1, 2, 3, 4}, {5, 6, 7} };

Numbers is now an array with itself holding two arrays as its elements.
To access the elements of the Numbers array,we specify two indexes: one for the array, and other for the elements inside that array. Following example accesses the third element (index-2) in the second array (index-1) of Numbers: Numbers अब अपने तत्वों के रूप में दो सरणियों/त्रमों को धारण करने वाली एक सरणी/त्रम है।
Numbers सरणी/त्रम के तत्वों तक पहुंचने केलिए, हम दो अनुक्रमणिका प्रदान करते हैं: एक सरणी(त्रम) केलिए, और दूसरा उस सरणी(त्रम) के अंदर तत्वों केलिए। निम्नलिखित उदाहरण संख्याओं के दूसरे सरणी (अनुक्रमणिका-1) में तीसरे तत्व (अनुक्रमणिका-2) तक पहुँचता है:

Example:

उधारण:


अंक [][] Numbers = { {1, 2, 3, 4}, {5, 6, 7} }; अंक x = Numbers[1][2]; तंत्र.बाहर.प्रिंट(i);

We can also use a केलिए(for) loop inside another केलिए(for) loop so as to get the elements of our two-dimensional array (we again have to point to the two indexes): हम अपने दो-आयामी सरणी/त्रम के तत्वों को प्राप्त करने केलिए एक केलिए(for) लूप का उपयोग कर सकते हैं (हमें फिर से दो इंडेक्स को इंगित करना होगा):
