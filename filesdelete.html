 Delete Files| फाइले नष्ट करे :

To delete a file , use the नष्ट() विधि (delete() method) : किसी फ़ाइल को हटाने के लिए, नष्ट() विधि (delete() विधि) का उपयोग करें:

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ कोशिश { फाइल मेरीवस्तु = नई फाइल ("फाइलकानाम.txt");//या पूर्ण रास्ता C:\\Users.. अगर ( मेरीवस्तु.नष्ट() ){ तंत्र.बाहर.प्रिंट(" फाइल नष्ट कर दी गई | "); } या { तंत्र.बाहर.प्रिंट(" फ़ाइल को नष्ट करने में विफल हुए । " } } पकड़े (त्रुटि त ){ तंत्र.बाहर.प्रिंट("कुछ गलती या त्रुटि पाई गई है | "); तंत्र.बाहर.प्रिंट( त ); } } }



 Delete a Folder | एक फ़ोल्डर नष्ट करें :

We can also delete a folder. However, it must be empty : हम किसी संचिका संयोजक/फोल्डर को नष्ट/डिलीट भी कर सकते हैं। हालाँकि, यह खाली होना चाहिए :

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ फाइल मेरीवस्तु = नई फाइल ("C:\\Users\\username\\Test"); अगर ( मेरीवस्तु.नष्ट() ){ तंत्र.बाहर.प्रिंट(" संचिका संयोजक/फोल्डर नष्ट कर दी गई | "); } या { तंत्र.बाहर.प्रिंट(" संचिका संयोजक/फोल्डर को नष्ट करने में विफल हुए । " } } }
