 Type Casting|पार्कर ढालना|प्रकार बदलना:

Type casting occurs when you assign a value of one of primitive data type to another data type.
There are two types of casting:
  • Widening Casting(done automatically) - converting a smaller type to a larger type size
    बाइट->छोटा->char->इंट->लंबा->फ्लोट->बड़ा
  • Narrowing Casting (manually) - converting a larger type to a smaller size type
    बड़ा ->फ्लोट -> लंबा->इंट-> char -> छोटा -> बाइट
 पार्कर ढालना तब होता है जब आप किसी अन्य डेटा प्रकार के लिए आदिम डेटा प्रकार में से एक का मान(value) प्रदान करते हैं।
ढालना दो प्रकार की होती है:
  • चौढाते या बढ़ते हुए ढालना - एक छोटे प्रकार को बड़े प्रकार के आकार प्रदान स्वचालित रूप से किया जाएगा
    बाइट->छोटा->char->इंट->लंबा->फ्लोट->बड़ा नैरोइंग
    (स्वचालित रूप से किया जाएगा )
  • संकुचन करते हुए ढालना- एक बड़े प्रकार को छोटे आकार के प्रकार में परिवर्तित करना
    बड़ा ->फ्लोट -> लंबा->इंट-> char -> छोटा -> बाइट
    (आप को करना होगा)
