Similarly most programming languages have their own syntaxing to write software programs. Java,Python,C,VisualBasic,One4 and all other programming languages too have their own syntaxing. इसी तरह एक प्रोग्रामिंग भाषा का सॉफ्टवेयर प्रोग्राम लिखने केलिए अपना सिंटैक्स होता है। वन4(one4), जावा(Java), पायथन(Python), सी(C), विजुअल बेसिक(V.B.) अन्य सभी प्रोग्रामिंग भाषाओं का अपना वाक्य विन्यास (syntax)होता है।

// C code to print hello world #include <stdio.h> int main() { printf("hello, world!"); return 0; } //JavaScript code to print hello world console.log('hello world'); //Perl code to print hello world #!/usr/bin/perl # Modules used use strict; use warnings; # Print function print("hello world\n"); //Python code to print hello world print("hello world")

ध्यान दे:

यहां आपको इतनी सारी अलग-अलग प्रोग्रामिंग भाषाओं के अलग-अलग वाक्यविन्यास(syntaxing) के बारे में चिंता करने की आवश्यकता नहीं है, क्योंकि किसी एक भाषा को सीखने के बाद आपकेलिए अन्य भाषाओं को सीखना बहुत आसान होगा।
इसके अलावा उनमें से अधिकांश के वाक्यविन्यास(syntaxing) में बहुत मामूली बदलाव होते हैं। (उपरोक्त भाषाएं वह हैं जो समान वाक्यविन्यास(syntaxing) साझा नहीं करती हैं इसलिए तनाव लेने की कोई आवश्यकता नहीं है)।
