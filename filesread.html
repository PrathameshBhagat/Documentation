In the previous chapter, we learned how to create and write to a file.

In the following example, we use the स्कैनर वर्ग (Scanner class) and हैअगलीरेखा विधि (hasNextLine() methord) to read the contents of the text file we created in the previous topic : पिछले अध्याय में, हमने सीखा कि फाइल कैसे बनाई जाती है और उसमें कैसे लिखा जाता है। निम्नलिखित उदाहरण में, हम स्कैनर वर्ग (Scanner class) और हैअगलीरेखा विधि (hasNextLine() methord) का उपयोग उस फ़ाइल की सामग्री को पढ़ने के लिए करते हैं जिसे हमने पिछले विषय/अध्याय में बनाया था:

Example:
