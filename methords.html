  Methords|विधियाँ:

A method(विधि) is a block of code which only runs when it is called.
We can pass various data values(called parameters) into a method.
In programming methords(also know as functions) are used to perform certain actions/tasks .
Methords are used to reuse code i.e define the code once, and use it many times. एक विधि कोड का एक ब्लॉक/संवर्ग है जो केवल तभी चलता है जब इसे कहा जाता है।
हम एक विधि में विभिन्न डेटा मान (पैरामीटर कहा जाता है) पास कर सकते हैं।
प्रोग्रामिंग मेथर्ड्स में (जिसे < के रूप में भी जाना जाता है) b>फ़ंक्शंस) का उपयोग कुछ कार्यों/कार्यों को करने के लिए किया जाता है।
तरीकों का उपयोग कोड का पुन: उपयोग करने के लिए किया जाता है यानी कोड को एक बार परिभाषित करें, और इसे कई बार उपयोग करें।

  Create a Method|एक विधि बनाएँ

A method(विधि) must be declared within a कक्षा(class).A method is defined with the name of the method, followed by parentheses ().we have some pre-defined methods, such as तंत्र.बाहर.प्रिंट(), but we can also create our own methods to perform certain tasks/actions : एक विधि को (class) के भीतर घोषित किया जाना चाहिए। एक विधि को उसके के नाम से परिभाषित किया जाता है, इसके बाद कोष्ठक () होते हैं। हमारे पास कुछ पूर्व-परिभाषित विधियाँ हैं , जैसे कि तंत्र.बाहर.प्रिंट(), लेकिन हम कुछ कार्यों को करने के लिए अपनी नई विधि भी बना सकते हैं:

Creating a method named मेरीविधि inside मुख्य class:

उदाहरण:

मुख्य समूह या कक्षा(class) के अंदर मेरीविधि नामित एक विधि बनाना:
सार्वजनिक समूह मुख्य { स्थिर रिक्त मेरीविधि(){ //संहित/कोड जो निष्पादित किया जाना है //code to be executed } }

Example Explained:
  • मेरीविधि() is the name of the method(विधि)
  • स्थिर means that the method belongs to the मुख्य which is a समूह(class) and not an object of मुख्य . We will learn more about objects and how to access methods through objects later in this tutorial.
  • रिक्त means that this method(विधि) does not have a return value.We will learn more about return values later in this chapter. उदाहरण समझे :
    • मेरीविधि(), विधि का नाम है
    • स्थिर का अर्थ है कि विधि "मुख्य" विधि से संबंधित है जो एक समूह(class) है और "मुख्य" का हिस्सा/वस्तु (object) नहीं है। हम बाद में हिस्सा/वस्तु (object) के बारे में और उनके माध्यम से उनकी विधियों तक पहुंचने के तरीके के बारे में अधिक जानेंगे।
    • रक्त का अर्थ है कि इस विधि (विधि) का वापसी मूल्य नहीं है। हम आगे इस अध्याय में बाद में वापसी मूल्यों के बारे में अधिक सीखेंगे।
