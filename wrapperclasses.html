 Wrapper Classes|आवरण वर्ग |ओढ़ने/लपटने वाले वर्ग :

आवरण वर्ग (Wrapper classes) provide a way to use आदिम डेटा प्रकार(primitive data types) (अंक,बाइट , etc..) as वस्तु (objects).

The table below(yet to be built ) shows the आदिम डेटा प्रकार(primitive data types) and the equivalent आवरण वर्ग (Wrapper classes). आवर्त वर्ग (Wrapper classes) आदिम डेटा प्रकार(primitive data type) (अंक, बाइट , आदि..) वस्तु (object) के रूप में उपयोग करने का एक तरीका प्रदान करते हैं।

नीचे दी गई तालिका (अभी तक बनाई जानी है) आदिम डेटा प्रकार (primitive data types) और उनकी आवर्त वर्ग (Wrapper classes) दिखाती है।
