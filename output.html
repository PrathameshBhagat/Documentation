 Print :

We learned from the previous topic that you can use the println() method to output values or print text in our programming language: हमने पिछले अध्याय से सीखा है कि आप हमारी प्रोग्रामिंग भाषा में मूल्यों को लिख के दिखाने के लिए (आउटपुट या प्रिंट करने के लिए) println() विधि का उपयोग कर सकते हैं:

सार्वजनिक कक्षा Main { सार्वजनिक स्थिर रिक्त जरूरी (डोर [] a){ तंत्र.बाहर.प्रिंट("Hello World"); } }

We can add as many println() methods as you want. Note that it will add a new line for each method: आप जितनी चाहें उतनी println() विधियाँ जोड़ सकते हैं. ध्यान दें कि यह प्रत्येक विधि के लिए एक नई लाइन जोड़ देगा:

सार्वजनिक कक्षा Main { सार्वजनिक स्थिर रिक्त जरूरी (डोर [] a){ तंत्र.बाहर.प्रिंट("Hello World"); तंत्र.बाहर.प्रिंट("Iam learning a language"); तंत्र.बाहर.प्रिंट("This is awesome!"); } }

 Double Quotes|दो बोलियां/कोट्स/उद्धरण| " "

When you are working with text, it must be wrapped inside double quotations marks " ".
If you forget the double quotes, an error occurs: जब आप (screen पर) लेखन या लिखने का कार्य कर रहे हों, तो इसे दोहरे उद्धरण " " चिह्नों के अंदर लपेटा जाना चाहिए.
यदि आप दोहरे उद्धरण भूल जाते हैं, तो एक त्रुटि/गलती/एरर(Error) उत्पन्न होती है:

This sentence will work : ये काम नहीं करेगा :
तंत्र.बाहर.प्रिंट("This will work!|ये काम करेगा");
This sentence will produce an ERROR : ये काम नहीं करेगा और त्रुटि (Error) देगा :
तंत्र.बाहर.प्रिंट(This sentence will produce an ERROR);

  The Print() Method| मुद्रण() विधि

There is also a print() method, which is similar to println().
The only difference is that it does not insert a new line at the end of the output: एक print() विधि भी है, जो println() के समान है।
दोंनो मे एकमात्र अंतर यह है कि यह आउटपुट के अंत में एक नई लाइन सम्मिलित नहीं करता है या जोड़ता नहीं है :

तंत्र.बाहर.प्रिंट("Hello World! "); तंत्र.बाहर.प्रिंट("I will print on the same line.");

Note :

Note that we add an extra space (after "Hello !" in the example above), for better readability.

ध्यान दे:

ध्यान दें कि हम बेहतर पठनीयता(पढ़ने की क्षमता) के लिए एक अतिरिक्त स्थान (ऊपर दिए गए उदाहरण में "hello! " के बाद जगह(स्पेस) छोड़ते हैं)|

 Print Numbers :

You can also use the println() method to print numbers. However, unlike text, we don't put numbers inside double quotes: आप संख्याओं को मुद्रित करने के लिए println() विधि का भी उपयोग कर सकते हैं।
हालाँकि,सादाहरण लेखन के विपरीत, हम संख्याओं को दोहरे उद्धरणों या कोट्स " " के अंदर नहीं डालते हैं:

तंत्र.बाहर.प्रिंट(8); तंत्र.बाहर.प्रिंट(798); तंत्र.बाहर.प्रिंट(50000);

You can also perform mathematical calculations(add, substract etc.) inside the println() method: आप println() विधि के अंदर गणितीय गणना (जोड़ना,घटना आदि) भी कर सकते हैं:

तंत्र.बाहर.प्रिंट(8+4); तंत्र.बाहर.प्रिंट(15-6); तंत्र.बाहर.प्रिंट(5*6); तंत्र.बाहर.प्रिंट(36/6); तंत्र.बाहर.प्रिंट(50%3);

Note :

You can do these operations/calculations in the print() and in println() method difference is that it won add a new line but not write it in same line.
Also you can add multiple println() and print() methords one after other as we did before.

ध्यान दे:

आप print() और println() विधि में समान तरीके से गणना कर सकते हैं, अंतर यह है कि यह एक नई पंक्ति नहीं जोड़ेगा बल्कि इसे उसी पंक्ति में लिखेगा या कहे जोड देगा । इसके अलावा, आप एक के बाद एक कई println() और print() विधियों को जोड़ सकते हैं जैसा कि हमने पहले किया था।
