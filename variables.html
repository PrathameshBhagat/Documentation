 Variables :

Variables are containers for storing data values.There are different types of variables, for example:
  • डोर - stores text, such as "Hello". String values are surrounded by double quotes
  • int - stores integers (whole numbers), without decimals, such as 123 or -123
  • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
  • वर्ण - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
  • boolean - stores values with two states: true or false.
आकड़ों या डेटा मानों को संग्रहीत(store) करने के लिए कंटेनर या कहे एक धारक/डिब्बे का काम करता हैं। ये विभिन्न प्रकार के होते हैं, उदाहरण के लिए:
  • डोर - एक रेखा मे लिखे चीजों को संग्रहीत(store) करता है। स्ट्रिंग मान दोहरे उद्धरणों " "से घिरे हुए होते हैं।
    जैसे :"hello"
  • int - दशमलव(डेसीमल) के बिना पूर्णांक(पूर्ण संख्या/Whole numbers) संग्रहीत(store) करता है। (decimal without whole numbers)
    जैसे : 123 या -123
  • float - फ़्लोटिंग पॉइंट संख्याओं को दशमलव के साथ संग्रहीत(store) करता है ।(पूर्णांक के साथ दशमलव)
    जैसे : 19.99 या -19.99
  • वर्ण - एक वर्ण(character) को संग्रहीत करता है । वर्ण एक उद्धरणों ' ' से घिरे हुए होते है ।
    जैसे 'a' या 'b'
  • boolean -ये केवल दो चीजे को संग्रहीत(store) करता है।
    जैसे : सही या गलत।

 Declaring (Creating) Variables|Variable घोषित करना(या बनाना)

To create a variable, you must specify the type and assign it a value: variable बनाने के लिए, आपको प्रकार स्पष्ट करना होगा और इसे मूल्य(वैल्यू/value) या अंक देना करना होगा:

Syntax:

type variableName = value;

वाक्य रचना:

प्रकार नाम = मूल्य (वैल्यू);


Where type is one of variable types discussed earlier (such as int or डोर), and variableName is the name of the variable (such as x or name). The equal sign =is used to assign values to the variable.
To create a variable that should store text, look at the following example: जहां प्रकार पहले चर्चा किए गए प्रकारों में से एक है (जैसे ईंट या डोर), और variablename नाम variable का नाम है (जैसे x या name)। बराबर चिह्न = variable को मूल्य(value) प्रदान करने के लिए उपयोग किया जाता है।
अगर हम टेक्स्ट(text) संग्रहीत(store) करने के लिए variable बनाना कहते है तो निम्न उदाहरण का प्रयोग कर सकते है :

Example:

Create a variable named as name of type डोर and assign/give it the value "Ram":

उदाहरण:

डोर प्रकार के name के रूप में नामित एक variable बनाएं और इसे "Ram" मान(value) प्रदान करें
डोर name ="Ram"; तंत्र.बाहर.प्रिंट(name);

If you want to create a variable that should store a number, look at the example below: यदि आप एक variable बनाना चाहते हैं जिसे एक संख्या संग्रहीत करनी चाहिए, तो नीचे दिए गए उदाहरण को देखें:

Example:

Create a variable called number of type इंट and assign it the value 45:

उदाहरण:

इंटप्रकार की संख्या नामक एक variable बनाएं और इसे मान(वैल्यू) 45 असाइन करें:
इंट number =45; तंत्र.बाहर.प्रिंट(number);

We can also declare a variable and assign the value later(without assigning value previously) : हम एक variable घोषित कर सकते हैं और बाद में मान(value) प्रदान कर सकते हैं (पहले मूल्य/वैल्यू प्रदान किए बिना) :

Example:

उदाहरण:


इंट number; number =45; तंत्र.बाहर.प्रिंट(number);

Note that if you assign a new value to an existing variable, it will overwrite the previous value: ध्यान दें कि यदि आप किसी मौजूदा variable को कोई नया मूल्य प्रदान करते हैं, तो यह पिछले मूल्य(value) प्रदान को अधिलेखित कर देगा(बदल देगा ):

Example:

Change the value of number 45 to 50:

उदाहरण:

संख्या 45 का मान/वैल्यू 50 में परिवर्तित करें
इंट number=45; number =50; तंत्र.बाहर.प्रिंट(number);


 Final Variables|

If you don't want others (or yourself) to overwrite existing values, use the अंतिम keyword (this will declare the variable as "final" or "constant", which means unchangeable and read-only) यदि आप नहीं चाहते हैं कि कोई अन्य (या स्वयं आप) मौजूदा variable को अधिलेखित करें(या ना बदले), तो अंतिम का उपयोग करें (यह variable को "अंतिम" के रूप में घोषित करेगा, जिसका अर्थ है अपरिवर्तनीय और केवल पढ़ने के लिए है और इसे बदल नहीं जा सकता):

Example:

उदाहरण:


अंतिम number=45; number =50;//Error produced


 Other Types|बाकी प्रकार के variables

A demonstration of how to declare variables of other types: अन्य प्रकार के variable को इस तरीके से धोषित करे:

Example:

उदाहरण:


इंट number =8; फ्लोट floatnumber =5.99फ; वर्ण letter='a'; बूलियन bool=सच; डोर text ="I m happy";

 Display Variables|Variable प्रदर्शित करे

The println() method is often used to display variables.
To combine both text and a variable, use the + character: println() विधि(methord) का उपयोग अक्सर variable प्रदर्शित या कहे प्रिन्ट करने के लिए किया जाता है।
टेक्स्ट और variable दोनों को संयोजित करने के लिए,(जोड़ने के लिए)+ वर्ण का उपयोग करें:

Example:

उदाहरण:


डोर text ="I m happy"; तंत्र.बाहर.प्रिंट("Hello!"+text);

You can also use the +character to add a variable to another variable: आप किसी अन्य variable में एक दूसरा variable जोड़ने के लिए + का भी उपयोग कर सकते हैं:

डोर text ="Hello!"; डोर text1 ="I m happy"; तंत्र.बाहर.प्रिंट(text+text1);

For numeric values, the + character works as a mathematical operator (notice that we use int (integer) variables here): संख्यात्मक मानों/मूल्यों(values) के लिए, + वर्ण गणितीय ऑपरेटर के रूप में काम करता है (ध्यान दें कि हम यहां इंट(पूर्णांक) चर का उपयोग करते हैं):

इंट x =5; इंट y =7; तंत्र.बाहर.प्रिंट(x+y);

From the example above, you can expect:
  • x stores the value 5
  • y stores the value 7
  • Then we use the println() method to display the value of x + y, which is 11
 ऊपर दिए गए उदाहरण से, आप उम्मीद कर सकते हैं:
  • x मान 5 संग्रहीत करता है
  • वाई मूल्य 6 संग्रहीत करता है
  • फिर हम x + y के मूल्य को प्रदर्शित करने के लिए println() विधि(method) का उपयोग करते हैं, जो 11 है|

 Declare Multiple Variables|अनेक Variables घोषित करे

To declare more than one variable of the same type, you can use a comma-separated list: एक ही प्रकार के एक से अधिक variable घोषित करने के लिए, आप अल्पविराम ,से अलग की गई सूची का उपयोग कर सकते हैं(comma-separated list):

इंट x =5; इंट y =7; इंट z =50; तंत्र.बाहर.प्रिंट(x+y+z); //or you can use इंट x=5,y=7,z=50; तंत्र.बाहर.प्रिंट(x+y+z);

 One Value to Multiple Variables|एकाधिक Variables के लिए एक मान/मूल्य(वैल्यू)

You can also assign the same value to multiple variables in one line: आप एक पंक्ति में एकाधिक variable के लिए एक ही मूल्य(value) भी प्रदान कर सकते हैं:

इंट x ,y,z; x=y=z=50; तंत्र.बाहर.प्रिंट(x+y+z);//150

 Identifiers|पहचानकर्ता|variable के नाम

All variables must be identified with unique names.
These unique names are called identifiers.They can be short names (like x and y) or more descriptive names (age, sum, totalVolume).

Note: It is recommended to use descriptive names in order to create understandable and maintainable code: सभी variable को अलग नामों(अद्वितीय नामो) से पहचाना जाना चाहिए। इन अद्वितीय नामों को पहचानकर्ता(identifiers) कहा जाता है। पहचानकर्ता छोटे नाम (जैसे x और y) या अधिक वर्णनात्मक नाम (age, sum, totalVolume) हो सकते हैं।

ध्यान दे: समझने योग्य और रखरखाव योग्य कोड बनाने के लिए वर्णनात्मक नामों का उपयोग करना बहतर माना जाती है:

इंट speedkmperhour=50;//This is good इंट s=50;//ok but not so easy to understand what s is

The general rules for naming variables are:
  • Names can contain letters(a,b,c...z or A,B,C...Z) , digits, underscores, and dollar signs
  • Names must begin with a letter
  • Names can also begin with $ and _
  • Names are case sensitive ("myVar" and "myvar" are different variables)
  • Reserved words (like Java keywords, such as int or boolean or php keywords like php ,list would be later specified)cannot be used as variable names
  • Names should preferably start with a lowercase letter and it can never contain whitespace
 variable के नाम लिए सामान्य नियम हैं:
  • नामों में अक्षर (केवल a,b,c...z और A,B,C...Z),अंक (केवल 1,2,3.. ), अंडरस्कोर_ और डॉलर$ के संकेत हो सकते हैं
  • नाम एक अक्षर से ही शुरू होना चाहिए (केवल a,b,c...z और A,B,C...Z)
  • नाम किसी लोअरकेस अक्षर से प्रारंभ हो तो बेहतर होगा और इसमें कभी भी श्वेतस्थान(whitespace)   नहीं हो सकता
  • नाम भी $ और _ के साथ शुरू कर सकते हैं
  • नाम केस(case) संवेदी हैं ("vaVar" और "vavar" अलग-अलग variables हैं)
  • आरक्षित शब्द(reserved keywords) जैसे Java के आरक्षित शब्द या PHP के आरक्षित शब्द या , उधारण के लिए int,php,php या अगर का उपयोग नाम के रूप में बिलकुल नहीं किया जा सकता है |
