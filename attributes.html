 Class Attributes|कक्षाओं/वर्गों की वस्तुओं के विशेषताएँ:

In the previous topic, we used the term "variable" for x(अ) in the example. It is actually an गुण/विशेषता(attribute) of the वर्ग(class). Or we could say that वर्ग गुण/वर्ग विशेषता(class attribute) are variables within a वर्ग(class): हमने पिछले उद्धरणों में x(अ) के लिए "वेरिएबल" शब्द का प्रयोग किया था । यह वास्तव में वर्ग(class) का एक गुण/विशेषता (attribute) है। या हम कह सकते हैं कि वर्ग गुण/वर्ग विशेषता (class attribute) एक वर्ग(class) के भीतर "वेरिएबल"/"चर" हैं:

Example:

Create a वर्ग/कक्षा(class) called "मुख्य" with two attributes(विशेषता): अ and आ

उदाहरण:

दो विशेषताओं के साथ "मुख्य" नामक एक वर्ग बनाएँ:अ and आ
कक्षा मुख्य { अंक अ =10; अंक आ =20; }

Note :

कक्षा is same as वर्ग we can any time write कक्षा in place of वर्ग and vice-verasa.

ध्यान दे:

वर्ग , कक्षा के समान है हम किसी भी समय वर्ग के स्थान पर कक्षा और कक्षा के स्थान पर वर्ग लिख सकते हैं ।



 Accessing Attributes|विशेषता/गुण तक पहुँचना

We can access विशेषता/गुण(attributes) by creating an वस्तु(object) of the वर्ग(class), and by using the dot syntax .:
The below given example will create an वस्तु(object) of the "मुख्य" वर्ग(class), with the name "मेरीवस्तु". We use the "अ" विशेषता(attribute) on the वस्तु(object) to print its value: हम वर्ग(class) की एक वस्तु(object) बनाकर और दशमलव या डॉट . का उपयोग करके विशेषता/गुण(attributes) तक पहुँच सकते हैं।:
नीचे दिए गया उदाहरण "मुख्य" वर्ग (वर्ग) की एक वस्तु(object) का निर्माण करेगा जिसका नाम "मेरीवस्तु" होगा | हम "मेरीवस्तु" नामक वस्तु की "अ" विशेषता को उसी के मूल्य को प्रदर्शित करने के लिए इस्तेमाल करेंगे:

Example:

Create an object called "मेरीवस्तु" and print the value of "अ":

उदाहरण:

"मेरीवस्तु" नामक वस्तु बनाएँ और "अ" का मान प्रिंट करें:
कक्षा मुख्य { अंक अ =10; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु = नई मुख्य(); तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }// 10: प्रदर्शित करेगा



 Modify Attributes|विशेषता/गुण को संशोधित करें

We can also modify attribute(विशेषता) values: हम विशेषता मानों (attribute values) को भी संशोधित कर सकते हैं:

Example:

Set the value of अ to 50:

उदाहरण:

"मेरीवस्तु" नामक वस्तु बनाएँ और "अ" का मान प्रिंट करें:
कक्षा मुख्य { अंक अ; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु = नई मुख्य(); मेरीवस्तु.अ=50; तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }// 50 :प्रदर्शित करेगा

Or override existing values: या मौजूदा मूल्यों को अवहेलना करें(बदल दे):

Example:

Change the value of "अ" to 50 :

उदाहरण:

"अ" के मान/मूल्य को 50 में बदलें:
कक्षा मुख्य { अंक अ=10 ; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु = नई मुख्य(); मेरीवस्तु.अ=50; तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }// 50 :प्रदर्शित करेगा

If we don't want the ability to override existing values, declare the attribute as अंतिम: यदि हम मौजूदा मूल्यों को अवहेलना करने(बदलने) की क्षमता नहीं चाहते हैं, तो विशेषता को अंतिम घोषित करें:

Example:

उदाहरण:
कक्षा मुख्य { अंतिम अंक अ=10 ; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु = नई मुख्य(); मेरीवस्तु.अ=50;//ये त्रुटि उत्पन्न करेगा:will generate an error तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }//cannot assign a value to a final/अंतिम variable:प्रदर्शित करेगा

Note :

The अंतिम keyword is useful when you want a variable to always store the same value, like PI (3.14159...).
The final keyword is called a "modifier". You will learn more about these in the Modifiers Chapter.

ध्यान दे:

अंतिम तब उपयोगी होता है जब आप चाहते हैं कि एक वेरीअबल/चर हमेशा समान मान(value) संग्रहीत करे, जैसे PI (3.14159...)।
अंतिमको "संशोधक" कहा जाता है। आप इनके बारे में अधिक जानकारी संशोधक अध्याय में जानेंगे।
