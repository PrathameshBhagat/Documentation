  Break|भंग:

We have already seen the भंग statement used in an earlier chapter. It was used to "jump out" of a बदले(switch) statement.
The भंग(break) statement can also be used to jump out of a loop. हम पहले के अध्याय में उपयोग किए गए भंग कथन को पहले ही देख चुके हैं। इसका उपयोग बदले कथन के "बाहर कूदने" केलिए किया गया था।
लूप से बाहर कूदने केलिए भीभंग (ब्रेक) कथन का उपयोग किया जा सकता है।

भंगExample:

Given example stops the loop when i is equal to 4:

भंगउदाहरण:
केलिए(अंक i = 0; i < 10; i++) { अगर (i == 4) { भंग; } तंत्र.बाहर.प्रिंट(i); }

 Continue|जारी :

The जारी(continue) statement breaks one iteration(inside the loop), if a specified/provided condition occurs it continues with the next iteration in the loop. जारी (जारी) कथन एक पुनरावृत्ति (लूप के अंदर) को तोड़ता है, यदि कोई निर्दिष्ट/प्रदान की गई स्थिति आती है तो यह लूप में अगले पुनरावृत्ति के साथ जारी रहती है।(केवल एक पुनरावृत्ति को रोक देगा पर आगे लूप को जारी रखेगा)।

Example:

उदाहरण:
केलिए(अंक i = 0; i < 10; i++) { अगर (i == 4) { जारी; } तंत्र.बाहर.प्रिंट(i); }

 Breakभंग and Continueजारी in Whileजब|भंगऔरजारी , जबमे :

We can also use भंग(break) andजारी(continue) in while loops: हम जबलूप के दौरान भंग औरजारीका भी उपयोग कर सकते हैं:।

भंगExample:

भंगउदाहरण:
अंक i = 0; जब(i < 5){ तंत्र.बाहर.प्रिंट(i); i++; अगर (i == 4) { भंग; } }

जारीExample:

जारीउदाहरण:
अंक i = 0; जब(i < 5){ तंत्र.बाहर.प्रिंट(i); i++; अगर (i == 4) { जारी; } }
