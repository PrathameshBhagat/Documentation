 For Loop|के लिए लूप:

When we exactly know how many times we want to loopor repeat through a block of code,we use the के लिए(for) loop instead of a जब(while) loop: जब हम जानते हैं कि वास्तव में हम कोड के ब्लॉक को कितनी बार लूप या दोहराना चाहते हैं, तो हम जब लूप के बजाय के लिए के लिए(केलिए) लूप का उपयोग करते हैं:

Syntax:

वाक्यविन्यास:
केलिए(statement 1; statement 2; statement 3) { // code block to be executed }

Statement 1 is executed(once) before the execution of the given code block.
Statement 2 defines/specifies the condition for executing the provided code block.
Statement 3 is executed(each time) after the given code block has been executed. Statement 1 को दिए गए कोड ब्लॉक के निष्पादन से पहले (एक बार) निष्पादित किया जाता है।
Statement 2 प्रदान किए गए कोड ब्लॉक को निष्पादित करने के लिए शर्त को परिभाषित/निर्दिष्ट करता है।
दिए गए कोड ब्लॉक को निष्पादित करने के बाद Statement 3 निष्पादित किया जाता है(प्रत्येक बार)।

Example:

Given example will print the numbers 0 to 4:

उदाहरण:

दिए गए उदाहरण संख्याओं को 0 से 4 तक मुद्रित/प्रिन्ट करेंगे:
केलिए(इंट i = 0; i < 5; i++) { तंत्र.बाहर.प्रिंट(i); }


Example explained :
Statement 1 sets a variable before the loop starts (इंट i = 0).
Statement 2 specifies the condition for the loop to run i.e. i must be less than 5. If the condition is found to be true, then the loop will start over again, if it is found to be false, the loop will end.
Statement 3 will increases a value (i++) every time the code block in the loop has been executed. उदाहरण समझे :
Statement 1 लूप शुरू होने से पहले एक variable सेट करता है (इंट i = 0)।
Statement 2 लूप को चलाने के लिए शर्त स्पष्ट करता है (5 से कम होना चाहिए)। यदि स्थिति सही पाई जाती है, तो लूप फिर से शुरू हो जाएगा, अगर यह गलत पाया जाता है, तो लूप समाप्त हो जाएगा।
हर बार लूप में कोड ब्लॉक निष्पादित होने पर Statement 3 एक संख्या (i++) बढ़ाएगा।

Example:

This example will only print even values between 0 and 10:

उदाहरण:

यह उदाहरण केवल 0 और 10 के बीच सम मान मुद्रित करेगा
केलिए(इंट i = 0; i <= 10; i = i + 2) { तंत्र.बाहर.प्रिंट(i); }

 For-Each Loop|हर एकके लिए लूप:

There is also a "for-each" (हर एकके लिए) loop, which is used exclusively to loop through elements in an array: एक , हर एकके लिए लूप भी है, जिसका उपयोग विशेष रूप array तत्वों के माध्यम से लूप करने के लिए किया जाता है:

Syntax:

वाक्यविन्यास:
केलिए(type variableName : arrayName) { // code block to be executed }

Example:

Given example outputs all elements in the cars array, using a हर एकके लिए ("for-each") loop:

उदाहरण:

दिए गए उदाहरण cars array में सभी तत्वों को प्रदर्शित करता है, एक हर एकके लिए ("प्रत्येक के लिए") लूप का उपयोग करके:
डोर[] cars = {"Volvo", "BMW", "Ford", "Mazda"}; केलिए(डोर i:cars) { तंत्र.बाहर.प्रिंट(i); }
