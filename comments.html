  Comments| टिप्पणियाँ:

Comments are the ones that can be used to explain code, and also to make it more readable. It can also be used to prevent execution of the code when testing alternative code. संहिता(कोड/code) को समझाने और इसे और अधिक पढ़ने लायक बनाने केलिए टिप्पणियों(कमेंट्स) का उपयोग किया जा सकता है। इसका उपयोग वैकल्पिक(ऑल्टर्नेट) संहिता(कोड/code) का परीक्षण करते समय निष्पादन को रोकने केलिए भी किया जा सकता है।

 Single-line Comments|एक पंक्ति टिप्पणियाँ|एक रेखा(लाइन) मे टिप्पणियाँ

Single-line comments start's with special characters i.e two forward slashes //.
Any text lying between // and the end of line is ignored by compiler(and will not be executed). एक पंक्ति टिप्पणियाँ(एक रेखा/लाइन टिप्पणियाँ) दो फ़ॉरवर्ड स्लैश // से शुरू होती हैं।
// और पंक्ति(लाइन) के अंत के बीच के किसी भी हिस्से को कंपाइलर(या कहे कंप्युटर) द्वारा अनदेखा किया जाता है।(निष्पादित किया जाएगा/पढ़ा नहीं जाएगा)

Example:

Given example uses a single-line comment before a line of code:

उदाहरण:

यह उदाहरण संहिता/कोड की एक पंक्ति से पहले एक पंक्ति(रेखा) की टिप्पणी का उपयोग करता है।
//ये एक टिप्पणी है This is a comment तंत्र.बाहर.प्रिंट("Hello");


This example uses a single-line comment after the end of a line of code: यह उदाहरण संहिता/कोड की एक पंक्ति के अंत में एक पंक्ति(रेखा) की टिप्पणी का उपयोग करता है।

Example:

उदाहरण:


तंत्र.बाहर.प्रिंट("Hello"); //ये एक टिप्पणी है , This is a comment

 Multi-line Comments|बहु पंक्ति टिप्पणियाँ

Multi-line comments start with a backslash and a * i.e./* and ends with */.
Any text appearing between /* and */ will be ignored by compiler and is not read/compiled. बहु-पंक्ति/रेखा टिप्पणियाँ /* से शुरू होती हैं और */ पर समाप्त होती हैं।
/* और */ के बीच किसी भी पाठ को संकलक(कम्पाइलर या कहे कंप्युटर) द्वारा अनदेखा कर दिया जाएगा।

Example:

Following example uses a multi-line comment i.e. a comment block (to explain the code)

उदाहरण:

यह उदाहरण संहिता/कोड की व्याख्या करने केलिए एक बहु-पंक्ति टिप्पणी (एक टिप्पणी ब्लॉक) का उपयोग करता है :
/* The code below will print the word Hello this is a comment block */ तंत्र.बाहर.प्रिंट("Hello");

Note :

Single or multi-line comments?
It is up to you which you want to use.Genrally , we use // for short/smaller comments, and /* */ for longer/larger. स

ध्यान दे:

एक या बहु-पंक्ति टिप्पणियाँ?
यह आप पर निर्भर है कि आप किसका उपयोग करना चाहते हैं। आम तौर पर, हम छोटी टिप्पणियों केलिए // का उपयोग करते हैं, और /* */ लंबे समय केलिए।
