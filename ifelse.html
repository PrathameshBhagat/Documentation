 Conditions and If Statements|शर्तें और अगर....या कथन:

The usual logical conditions from mathematics mostly used in programming are:
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than: a > b
  • Equal to a == b
  • Not Equal to: a != b
We can use the following conditions to carry out different actions for different decisions.We have the following conditional statements to use above conditions :
  • Use अगर to specify a block of code to be executed, if the specified condition is true
  • Use या so as to specify a block of code to be executed, if the same condition(in if) is false
  • Use या अगर to specify a new condition to test, if the first condition is false
  • Use बदलो(switch) to specify many alternative blocks of code to be executed
 प्रोग्रामिंग में ज्यादा तर उपयोग किए जाने वाले गणित से सामान्य तार्किक स्थितियां हैं:
  • कम है: a < b
  • से कम या बराबर: a < b
  • से अधिक है : a > b
  • से अधिक या बराबर है : a >= b
  • a के बराबर == b
  • बराबर नहीं:a != b

उपरोक्त शर्तों का उपयोग करने के लिए हमारे पास निम्नलिखित सशर्त कथन हैं :
  • निष्पादित(execute) किए जाने वाले कोड के एक ब्लॉक को निर्दिष्ट करने के लिए अगर का उपयोग करें या
  • निष्पादित किए जाने वाले कोड के ब्लॉक को निर्दिष्ट करने के लिए या का उपयोग करें, यदि वही स्थिति गलत है।

  • निष्पादित किए जाने वाले कोड के कई वैकल्पिक ब्लॉक निर्दिष्ट करने के लिए बदलो का उपयोग करे

 If |अगर

Use the अगर statement to specify a block of code to be executed if a condition comes or computes out to be true. यदि कोई शर्त आती है या सत्य होने की गणना करती है तो निष्पादित(execute) किए जाने वाले कोड के ब्लॉक को निर्दिष्ट/स्पष्ट(specify) करने के लिए अगर कथन का उपयोग करें।

Syntax:

वाक्य विन्यास(syntax):
अगर (/*condition*/){ //code to be executed if the given condition is true }

In the following example , we test two values to check if 21 is greater than 11 . If the condition is true, then print some text: निम्नलिखित उदाहरण में, हम यह जांचने के लिए दो मानों का परीक्षण करते हैं कि क्या 21 , 11 से अधिक है? यदि स्थिति सही है, तो कुछ प्रिन्ट करें:

Example:

उदाहरण:
अगर (21 > 11) { तंत्र.बाहर.प्रिंट("20 is greater than 18"); }

Example:

We not only this but can also test variables

उदाहरण:

हम variable का परीक्षण भी कर सकते हैं:
इंट x = 21; इंट y = 11; अगर (x > y) { तंत्र.बाहर.प्रिंट("x is greater than y"); }

In the foreseen example we used two variables, x and y, to check if x is greater than y (using the > operator). As x is 21, and y is 11, and we know that 21 is greater than 11, we print to the screen that "x is greater than y". उदाहरण में हमने दो variable, x और y का उपयोग यह जांचने के लिए किया कि x , y से अधिक है या नहीं (> ऑपरेटर का उपयोग करके)। जैसे कि x 21 है, और y 11 है, और हम जानते हैं कि 21 , 11 से अधिक है, हम स्क्रीन पर प्रिंट करते हैं कि "एक्स वाई से अधिक है"।


 The या Statement |याकथन

Use the या statement to specify a block of code to be executed if the given condition is false or is incorrect . यदि दी गई शर्त गलत है, तो निष्पादित(execute) किए जाने वाले कोड के ब्लॉक को निर्दिष्ट/स्पष्ट करने के लिए या कथन का उपयोग करें

Syntax:

वाक्य विन्यास(syntax):
अगर (/*condition*/){ // the code to be executed if the given condition is true } या { //the block of code to be executed if the condition is false }

Example:

उदाहरण:
इंट time = 21 ; अगर(time < 11 ) { तंत्र.बाहर.प्रिंट("Good day."); } या { तंत्र.बाहर.प्रिंट("Good evening."); } // Outputs "Good evening."

Example explainedIn the example , time (21) is greater than 11, so the condition is false. Because of this, we move on to the याcondition and print on the screen "Good evening". If the time was less than 11, the program would print "Good day". उदाहरण का मतलब उदाहरण में, time(21) 11 से अधिक है, इसलिए स्थिति झूठी है। इस वजह से, हम यास्थिति पर जाते हैं और "Good evening" स्क्रीन पर प्रिंट करते हैं। यदि समय 11 से कम होता था, तो तंत्र "Good day" प्रिंट करेगा।

 The या अगर Statement |या अगरकथन

Use the या अगर statement to specify a block of code to be executed if the first given condition is false or is incorrect . यदि पहली दी गई शर्त गलत है, तो निष्पादित(execute) किए जाने वाले कोड के ब्लॉक को निर्दिष्ट/ स्पष्ट करने के लिए या अगर कथन का उपयोग करें।

Syntax:

वाक्य विन्यास(syntax):
अगर (/*condition1 */){ // the code to be executed if the given condition1 is true } या अगर (/*condition2 */){{ //the block of code to be executed if the condition1 comes out false and condition2 is true }या { //the block of code to be executed if the condition1 comes out false and condition2 is also comes out false }


Example:

उदाहरण:
इंट time = 21 ; अगर(time < 10 ) { तंत्र.बाहर.प्रिंट("Good morning."); } या अगर(time < 20 ){ तंत्र.बाहर.प्रिंट("Good day."); } या { तंत्र.बाहर.प्रिंट("Good evening."); } // Outputs "Good evening."

Example explained : In the example, time (21) is greater than 10, so the first condition is false. The next condition i.e. in the या अगर statement, is also false, so we move on to the last या condition since condition1 and condition2 is both false - and hence print to the screen "Good evening".
But if the time was 14, our program would print "Good day." उदाहरण के बारे में : उदाहरण में, समय (21) 10 से अधिक है, इसलिए पहली स्थिति झूठी है। अगली शर्त अर्थात या अगर कथन में,वो भी गलत है, इसलिए हम अंतिम या के अंदर की शर्त पर आगे बढ़ते हैं क्योंकि condition1 और condition2 दोनों झूठी हैं - और इसलिए स्क्रीन पर प्रिंट करें "Good evening"।
लेकिन अगर समय 14 था, तो हमारा कार्यक्रम "Good day" प्रिंट करेगा।

 The Short Handया अगर |छोटा या अगरकथन

There is also a short way to do या अगर ,it consists of three operands.
It is used to replace multiple lines (or blocks) of code with a single line, and is commonly used to replace simple या अगरstatements (if the expressions are small): या अगर इस्तेमाल करने का एक छोटा तरीका भी है , इसमें तीन वस्तुए होती हैं।
इसका उपयोग संहिता /कोड की कई रेखाओं(या ब्लॉक) को एक पंक्ति के साथ बदलने के लिए किया जाता है, और आमतौर पर सरल या अगरकथनों को बदलने के लिए उपयोग किया जाता है (यदि अभिव्यक्ति छोटी है):

Syntax:

वाक्य विन्यास(syntax):
//variable = (condition) ? expressionTrue : expressionFalse; //variable = (शर्त) ? अभिव्यक्तिसत्य : अभिव्यक्तिगलत;

Example:

Instead of writing

उदाहरण:

ये लिखने के बजाय
इंट time = 21 ; अगर(time < 10 ) { तंत्र.बाहर.प्रिंट("Good day."); } या { तंत्र.बाहर.प्रिंट("Good evening."); }

We can simply write: आप बस लिख सकते हैं:
इंट time = 21 ; डोर result = (time < 10) ? "Good day." : "Good evening."; तंत्र.बाहर.प्रिंट(result);
