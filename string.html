 String | डोर :

These are used for storing text in them .
A डोर contains a collection of characters surrounded by double quotes " " : इनका उपयोग पाठ/लेखन(text) को संग्रहीत करने केलिए किया जाता है।
एक डोर में दोहरे उद्धरणों " " से घिरे वर्णों का एक संग्रह होता है :

Example:

Given example creates a variable of type डोर and assigns it a value:

दिया गया उदाहरण डोर प्रकार का एक variable बनाता है और इसे एक value प्रदान करता है:

 String Length|एक पंक्ति टिप्पणियाँ|एक रेखा(लाइन) मे टिप्पणियाँ

A डोर (String) , here is actually an वस्तु (object), which contain विधियाँ (methods) that can perform certain operations on डोर (strings). For example, the length of a string can be found with the लंबाई()विधि ("लंबाई()" method) : एक डोर (String) (यहाँ) वास्तव में एक वस्तु (object) है, जिसमें विधियाँ (methods) होती हैं जो डोर (String) पर कुछ संचालन कर सकती हैं। उदाहरण के लिए, लंबाई()विधि ("लंबाई()" method) से एक डोर (String) की लंबाई पाई जा सकती है:

Example:

उदाहरण:
डोर लेखन = "ABCDEFGHIJKLMNOPQRSTUVWXYZअआइईउऊऋलृएऐओऔअंअः"; तंत्र.बाहर.प्रिंट("डोर की लंबाई है : " + लेखन.लंबाई() );



 Finding a Character in a String| एक डोर में वर्णों को ढूँढना

The अनुक्रमांक विधि (indexOf()) method returns the अनुक्रमांक or index (the position) of the first occurrence of a specified text in a डोर (string) (including whitespace ): अनुक्रमांक विधि , एक डोर (string) में निर्दिष्ट पाठ/लेखन की पहली प्राप्ती/घटना का सूचकांक/अनुक्रमांक लौटाती है (खाली स्थानो को गिनके ) :

Example:

उदाहरण:
डोर लेख = "कृपया 'मुझे' ढूँढे "; तंत्र.बाहर.प्रिंट( लेखन.अनुक्रमांक("मुझे"));

Note :

Here the counting positions starts from zero.

0 is the first position in a string, 1 is the second, 2 is the third ... . .

And if it's a hindi text then all "matra"'s are also counted , like "ा ी ि े ै " and even a "halant" i.e. "्" or a half character in hindi also gets an index to itself .

As in above example a half ऋ i.e " ृ" in कृ gets an index along with ा in या , ू in मू , े in झे and ( ' ) all combine to give output 7 .(including other characters) . You can also search for "ृ" or े to find their positions/indexes , just keep in mind that indexes starts from 0 .

ध्यान दे:

यहाँ शून्य से पदों की गणना की जाती है।
0 एक डोर में पहलाम क्रमांक है, 1 दूसरा है, 2 तीसरा है ...

और यदि यह एक हिंदी लेख है तो सभी "मात्रा"ओं की भी गिनती की जाती है जैसे "ा ी ि ू ु े ै" और यहां तक की एक "हलंत" यानी "्" या हिंदी में आधा वर्ण भी खुद को एक सूचकांक प्राप्त कराता है ।

उपर्युक्त उदाहरण की तरह एक आधा ऋ अर्थात कृ में "ृ" को एक अनुक्रमांक मिलता है , "या" मे ा , मुझे का ू और े ,साथ ही ' सभी मिलकर नतीजा 7 देते हैं। (अन्य वर्णों सहित). आप स्थिति का पता लगाने के लिए "ृ", "ी" या "ँ" आदि को भी खोज सकते हैं बस ध्यान रखें की अनुक्रमांक 0 से ही शुरू होगा ।

 String Concatenation | दो डोर को जोड़े | डोर संघनन

The + operator can be used between डोर (strings) to combine them. This is called संघनन (concatenation): + चिन्ह का उपयोग डोर (strings) के बीच उन्हें संयोजित करने के लिए किया जा सकता है। इसे डोर संघनन कहते हैं :

Example:

उदाहरण:
डोर नाम = "रमेश"; डोर उपनाम = "बाबू"; तंत्र.बाहर.प्रिंट(नाम + " " + उपनाम );



WARNING! :

Here we use the + operator for both addition and संघनन (concatenation).

Numbers are added डोर (Strings) are संघनन (concatenated).

चेतावनी! :

यहाँ हम जोडने और संयोजन दोनों के लिए + संचालिका का प्रयोग करते हैं।

संख्या/नंबर जोड़े जाते हैं , डोर (Strings) का संघनन/श्रृंखलाबद्धनन होता हैं।


If we add two numbers( अंक .. ) , the result will be a number ( अंक .. ) : यदि हम दो संख्याओं (अंक ..) को जोड़ते हैं, तो परिणाम एक संख्या ( अंक ..) होगा :

Example:

उदाहरण:
अंक अ = 100 ; अंक आ = 200 ; अंक ब = अ + आ ;//ब 300 होगा जो एक अंक है (ना की 100200)



If we add two डोर (strings), the result will be a डोर (strings concatenation ) : यदि हम दो डोर (strings) जोड़ते हैं, तो परिणाम एक डोर संयोजन (strings concatination) होगा :

Example:

उदाहरण:
डोर अ = "100" ; डोर आ = "200" ; डोर ब = अ + आ ;// ब 100200 होगा जो एक डोर है (ना की 300)



If we add a number( अंक..) and a डोर (string), the result will be a डोर (strings concatenation ) : यदि हम एक संख्या और एक डोर (string) जोड़ते हैं, तो परिणाम एक डोर संयोजन (strings concatination) ही होगा :

Example:

उदाहरण:
डोर अ = "100" ; अंक आ = "200" ; डोर ब = अ + आ ;// ब 100200 होगा जो एक डोर है (ना की 300)




 Strings - Special Characters | स्ट्रिंग्स मे विशेष वर्ण

As a डोर (string) must be written within quotes" ", compiler will misunderstand this डोर (string), and generate a त्रुटि error : डोर (string) को उद्धरण-चिह्न्हो ( " " ) के भीतर लिखा जाना चाहिए, कंप्युटर/कंपाइलर इस डोर (string) को गलत समझेगा, और एक त्रुटि (error)देगा :

Example:

उदाहरण:
डोर लेख = "हम "भारत" देश के नागरिक है ";//त्रुटि / गलती देगा

The solution to avoid this problem, is to use the backslash escape character.

The backslash (\) escape character turns special characters into डोर (string) characters : इस समस्या से बचने का उपाय पलायनवादी वर्ण का इस्तेमाल है : \ वर्ण (backslash escape character) का उपयोग करना है।

बैकस्लैश (\) वर्ण विशेष वर्णों को डोर (स्ट्रिंग) में बदल देता है :

Escape characterपलायनवादी वर्ण Resultनतीजा Descriptionवर्णन
\"" Would insert a " एक " डाल देगा
\'' Would insert a ' एक ' डाल देगा
\\\ Would insert a \ एक \ डाल देगा
\nNew Lineएक नई रेखा Would insert a New Line एक नई रेखा डाल देगा
\rCarriage Returnवाहक पुर्नागमन Would insert a Carriage Returnएक वाहक पुर्नागमन डाल देगा
\tTab8 बाइट Would insert a Tabएक टैब डाल देगा
\bBackspaceपश्चीयन Would insert a Backspaceएक वर्ण काम कर देगा
\fForm Feedपृष्ट भरण Would insert a Form Feedएक पृष्ट भरण डाल देगा

The sequence \" inserts a double quote ( " ) in a डोर (string): \" ये वर्ण मिलके एक डोर (string) में दोहरा उद्धरण ( " ) सम्मिलित करता है:

Example:

उदाहरण:
डोर लेख = "हम \"भारत\" देश के नागरिक है "; // हम "भारत" देश के नागरिक है : लिख देगा



The sequence \' inserts a single quote ( ' ) in a डोर (string): \' ये वर्ण मिलके एक डोर (string) में एक उद्धरण ( ' ) सम्मिलित करता है:

Example:

उदाहरण:
डोर लेख = "हम \'भारत\' देश के नागरिक है "; // हम 'भारत' देश के नागरिक है : लिख देगा

Similarly other characters would give respective outputs. और आगे ऐसे ही बाकी वर्णों से संबंधित नतीजे प्राप्त होंगे |
