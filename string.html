 String | डोर :

These are used for storing text in them .
A डोर contains a collection of characters surrounded by double quotes " " : इनका उपयोग पाठ/लेखन(text) को संग्रहीत करने के लिए किया जाता है।
एक डोर में दोहरे उद्धरणों " " से घिरे वर्णों का एक संग्रह होता है :

Example:

दिया गया उदाहरण डोर प्रकार का एक variable बनाता है और इसे एक value प्रदान करता है:
डोर text ="Hello";

 String Length|डोर.लंबाई|डोर की लंबाई

A डोर is actually an object, which contain methods(विधि) that can perform certain operations on strings.For instance , the length of a string can be obtained with the .लंबाई() method: एक डोर वास्तव में एक वस्तु है, जिसमें विधियां (विधि) होती हैं जो तारों पर कुछ ऑपरेशन कर सकती हैं। , मान लें कि स्ट्रिंग की लंबाई .लंबाई() विधि के साथ प्राप्त की जा सकती है:

Example:

उदाहरण:
डोर txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"; तंत्र.बाहर.प्रिंट("The length of the txt string is: " + txt.लंबाई());

 More String Methods|बाकी डोर विधियाँ

There are many string methods available, for example toUpperCase() and toLowerCase(): कई स्ट्रिंग विधियां उपलब्ध हैं, उदाहरण के लिए अपरकेस() और लोअरकेस():

Example:

उदाहरण:


डोर txt = "Hello World"; तंत्र.बाहर.प्रिंट(txt.अपरकेस()); // Outputs "HELLO WORLD" तंत्र.बाहर.प्रिंट(txt.लोअरकेस());//hello world

 Finding a Character in a String| डोर में वर्ण ढूँढना

The अनुक्रम() method outputs the index (the position) of the first occurrence of a given text in a string(डोर) [including whitespace]: अनुक्रम() विधि डोर में किसी दिए गए पाठ(text) की पहली मौजूदगी का अनुक्रमणिका (अनुक्रमांक) प्रदान करती है:[व्हाइटस्पेस सहित]

Example:

उदाहरण:


डोर txt = "Please locate where 'locate' occurs!"; तंत्र.बाहर.प्रिंट(txt.indexOf("locate")); //7

Note :

Here the compiler counts positions starting from zero.
0 is the first position/index in a string(डोर), 1 is the second, 2 is the third.

ध्यान दे:

यहां कंपाइलर/कंप्युटर शून्य से स्थितियों/क्रमों की गणना करता है।
0 एक स्ट्रिंग (डोर) में पहला स्थान/सूचकांक है, 1 दूसरा है,फिर 2 तीसरा है।

 String Concatenation| डोर संयोजन|डोर जोड़ना :

The + operator is used between डोर elements (strings) to combine them. This is termed as concatenation: + चिन्ह का उपयोग डोर तत्वों (स्ट्रिंग्स) के बीच उन्हें संयोजित करने(एक साथ जोड़ने) के लिए किया जाता है। इसे संयोजन कहा जाता है:

Example:

उदाहरण:
डोर firstName = "John"; डोर lastName = "Doe"; तंत्र.बाहर.प्रिंट(firstName + " " + lastName);

Note :

Here ,we have added an empty text element (" ") to create a space between firstName and lastName when we print them.

ध्यान दे:

यहां, हमने एक खाली तत्व (" ") जोड़ा है ताकि जब हम उन्हें प्रिंट करते हैं तो पहले firstName और lastName नाम के बीच एक रिक्त स्थान बनाया जा सके।

 Special Characters |विशेष वर्ण :

As डोर (strings) must be written within quotes, compiler will misunderstand this डोर (strings), and generate an error: जैसा कि डोरको उद्धरणों(" ") के भीतर लिखा जाना चाहिए, कंप्युटर/कंपाइलर/तंत्र इस डोर (स्ट्रिंग्स) को गलत समझेगा, और एक त्रुटि उत्पन्न करेगा।

Example:

उदाहरण:
डोर txt = "We are the so-called "Vikings" from the north.";;

The solution so as to avoid this problem, is to make use of the backslash escape character.
The backslash (\) escape character changes special characters into string characters: इस समस्या से बचने के लिए समाधान, प्रतिक्षेप पलायन वर्ण (बैकस्लैश एस्केप character) का उपयोग करना है।
\ वर्ण विशेष वर्णों को डोरके वर्णों में परिवर्तित करता है
