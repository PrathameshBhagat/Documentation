 Date and Time | दिनांक और समय

We have built-in Date,Time वर्ग (classes), to work with the date and time which includes date and time classes. For example: दिनांक और समय के साथ काम करने के लिए हमारे पास दिनांक, समय वर्ग(classes) हैं । उदाहरण के लिए :

Classवर्ग Descriptionवर्णन
दिनांक Represents a date (year, montd, day (yyyy-MM-dd)) एक दिनांक का प्रतिनिधित्व करता है(साल ,महिना ,दिन (yyyy-MM-dd))
समय Represents a time (hour, minute, second and nanoseconds (HH-mm-ss-ns)) एक समय का प्रतिनिधित्व करता है(घंटा ,मिनिट ,सेकंड (HH-mm-ss-ns))
दिनांकऔरसमय Represents botd a date and a time (yyyy-MM-dd-HH-mm-ss-ns) दिनांक और समय का प्रतिनिधित्व करता है(yyyy-MM-dd-HH-mm-ss-ns)
दिनांकऔरसमयस्वरूपण Formatter for displaying and parsing date-time objects दिनांक-समय वस्तु को प्रदर्शित करने के लिए प्रारूप प्रदान करता है


 Display Current Date|वर्तमान दिनांक प्रदर्शित करे

To display the current date,use दिनांकवर्ग(class), and use its अभी() विधि (now() method) : वर्तमान दिनांक प्रदर्शित करने के लिए, दिनांकवर्ग(class) का उपयोग करें, और इसके अभी() विधि (now() method) का उपयोग करें:

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ दिनांक मेरिवस्तु = नई दिनांक.अभी();// दिनांक वर्ग की वस्तु बनाए तंत्र.बाहर.प्रिंट(मेरिवस्तु);//दिनांक प्रदर्शित करे } }//2023-01-15 लिख देगा (जो भी दिनांक होगी वो |)



 Display Current Time|वर्तमान समय प्रदर्शित करे

To display the current time ,we can use an वस्तु (object) of समय वर्ग (class), and use its अभी() विधि (now() method) : वर्तमान समय प्रदर्शित करने के लिए, हम समय वर्ग (class) का एक वस्तु (object) का उपयोग कर सकते हैं, और इसकी अभी() विधि (now() method) का उपयोग कर सकते हैं:

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ समय मेरिवस्तु = नई समय.अभी();// दिनांक वर्ग की वस्तु बनाए तंत्र.बाहर.प्रिंट(मेरिवस्तु);//दिनांक प्रदर्शित करे } }//16:32:42.955429 लिख देगा (जो भी समय होगी वो |)



 Display Current Date and Time|अभी का समय और दिनांक प्रदर्शित करे

To display the current date and time ,we can use an वस्तु (object) of दिनांकऔरसमय वर्ग(class), and use its अभी() विधि (now() method) : वर्तमान दिनांक और समय प्रदर्शित करने के लिए, हम दिनांकऔरसमय वर्ग(class) के एक वस्तु का उपयोग कर सकते हैं, और इसकी अभी() विधि (now() method) का उपयोग कर सकते हैं। :

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ दिनांकऔरसमय मेरिवस्तु = नई दिनांकऔरसमय.अभी();// दिनांक वर्ग की वस्तु बनाए तंत्र.बाहर.प्रिंट(मेरिवस्तु);//दिनांक प्रदर्शित करे } }//2022-12-09T16:32:42.955722 लिख देगा (जो भी दिनांक होगी वो फिर समय लिखेगा |)



 Formatting Date and Time | समय और दिनांक का स्वरूपण करे(प्रारूप प्रदान करे)

The "T" in the example above is used to separate the date from the time. We can use the दिनांकऔरसमयस्वरूपण (DateTimeFormatter) class with the रूप() विधि ("ofPattern()" method) and स्वरूप() विधि to format or parse दिनांकऔरसमय वस्तु (date-time objects). The following example will remove both the "T" and nanoseconds from the date-time: उपरोक्त उदाहरण में "T" का उपयोग दिनांक को समय से अलग करने के लिए किया जाता है। हम दिनांक-समय (दिनांकऔरसमय) की वस्तुओं को प्रारूपित/स्वरूपित या पार्स करने के लिए रूप() विधि ("ofPattern()" method) और स्वरूप() विधि के साथ दिनांकऔरसमयस्वरूपण वर्ग (DateTimeFormatter class) का उपयोग कर सकते हैं। निम्न उदाहरण (दिनांकऔरसमय) से "T" और नैनोसेकंड दोनों को हटा देगा:

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क){ दिनांकऔरसमय मेरिवस्तु = नई दिनांकऔरसमय.अभी(); तंत्र.बाहर.प्रिंट(" पहले : " + मेरिवस्तु); दिनांकऔरसमयस्वरूपण मेरिदूसरीवस्तु =दिनांकऔरसमयस्वरूपण.रूप("dd-MM-yyyy HH:mm:ss"); डोर मेरितीसरीवस्तु = मेरिदूसरीवस्तु.स्वरूप(myFormatObj); तंत्र.बाहर.प्रिंट(" अब : " + मेरितीसरीवस्तु ); } }//पहले : 2022-12-09T16:32:42.955627 //अब : 09-12-2022 16:32:42 // लिख देगा (जो भी आभी की दिनांक , समय हो वो लिखेगा |)

The रूप() विधि ("रूप()" method) method accepts all sorts of values, if you want to display the date and time in a different format. For example: यदि आप दिनांक और समय को भिन्न स्वरूप में प्रदर्शित करना चाहते हैं, तो रूप() विधि ("रूप()" method) सभी प्रकार के मानों को स्वीकार करती है। उदाहरण के लिए :

Valueमूल्य/मान Exampleउद्धारण
yyyy-MM-dd"1988-09-29"
dd/MM/yyyy"29/09/1988"
dd-MMM-yyyy"29-Sep-1988"
E, MMM dd yyyy"Tdu, Sep 29 1988"
