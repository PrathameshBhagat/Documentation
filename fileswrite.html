 Create a File| फाइल/संचिका/दस्तावेज़ बनाए :

To create a file, you can use the फाइलबनाए विधि ("createNewFile" method). This method returns a दोहरा (Boolean) value: सच (true) if the file was successfully created, and झूठ (false) if the file already exists. Note that the विधि (method) is enclosed in a कोशिश .... पकड़े (try...catch) block. This is necessary because it throws an error , if the file cannot be created for some reason : फ़ाइल बनाने के लिए, आप फाइलबनाए विधि ("createNewFile" methord) का प्रयोग कर सकते हैं। यह विधि एक दोहरा (boolean) मान लौटाती है: सच (true) यदि फ़ाइल सफलतापूर्वक बनाई गई थी, और झूठ (false) यदि फ़ाइल पहले से मौजूद है। ध्यान दें कि विधि कोशिश .... पकड़े (try...catch) में संलग्न होनी चाहीए है। यह आवश्यक है क्योंकि यह एक त्रुटि फेंकता है, अगर फ़ाइल किसी कारण से नहीं बनाई जा सकती तो :

Example:

उदाहरण:
वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ कोशिश { फाइल मेरीवस्तु = नई फाइल ("फाइलकानाम.txt"); अगर ( मेरीवस्तु.फाइलबनाए() ){ तंत्र.बाहर.प्रिंट("फाइल बनाई गई : " + ); } या { तंत्र.बाहर.प्रिंट("फाइल पहले से मौजूद है | "); } } पकड़े (त्रुटि त ){ तंत्र.बाहर.प्रिंट("कुछ गलती या त्रुटि पाई गई है | "); } } }


To create a file in a (at a) specific directory (requires permission), specify the path of the file and use double backslashes to escape the "\" character (for Windows). On Mac and Linux you can just write the path, like: /Users/name/फाइलकानाम.txt एक विशिष्ट निर्देशिका (directory) (जिसके लिए अनुमति की आवश्यकता होती है) में एक फ़ाइल बनाने के लिए, फ़ाइल का पथ निर्दिष्ट करें और "\" वर्ण से बचने के लिए डबल बैकस्लैश का उपयोग करें (Windows के लिए)। मैक (Mac) और लिनक्स (Linux) पर आप सिर्फ पाथ लिख सकते हैं, जैसे: /Users/name/फाइलकानाम.txt


Example:

उदाहरण:
फाइल मेरीवस्तु = नई फाइल("C:\\Users\\MyName\\फाइलकानाम.txt");



 Write To a File| फाइल/संचिका/दस्तावेज़ मे लिखे :

In the following example, we use the फाइललेखक वर्ग (FileWriter class) together with its लिखे() विधि (write method) to write some text to the file we created in the example above. Note that when you are done writing to the file, you should close it with the बंद() विधि (close method): निम्नलिखित उदाहरण में, हम फाइललेखक वर्ग (FileWriter class) का उपयोग इसके लिखे() विथ (write method) के साथ करते हैं ताकि हमारे द्वारा बनाई गई फ़ाइल में कुछ पाठ लिखा जा सके। ऊपर का उदाहरण। ध्यान दें कि जब आप फ़ाइल में लिखना समाप्त कर लें, तो आपको इसे बंद() विधि (close method) से बंद करना चाहिए :

Example:

उदाहरण:
