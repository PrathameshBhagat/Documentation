 File| फाइल | संचिका | दस्तावेज़ :

File handling is an important part of any application or software .

We have several methods for creating, reading, updating, and deleting files. फ़ाइले संभालना किसी भी एप्लिकेशन ,सॉफ़्टवेयर या कार्यक्रम का एक महत्वपूर्ण हिस्सा होता है।

फ़ाइलें बनाने, पढ़ने, सुधारने और मिटाना के लिए हमारे पास कई विधियां हैं।



  File Handling| फ़ाइले संभालना

The फ़ाइल वर्ग (File class) , allows us to work with files.

To use the it , create a वस्तु (object) of the वर्ग (class), and specify the filename or directory name : फ़ाइल वर्ग (File class) , हमें फ़ाइलों के साथ काम करने योग्य बनाता है।

इसका उपयोग करने के लिए, फ़ाइल वर्ग (File class) की एक वस्तु (object) बनाएँ, और फ़ाइल का नाम या निर्देशिका नाम (directory name) स्पष्ट करें :

Example:

उदाहरण:
फाइल मेरीवस्तु = नई फाइल("फाइलकानाम.txt");//फाइलकानाम दे
