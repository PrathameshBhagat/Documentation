The Math class has many methods that allows us to perform mathematical tasks गणित कक्षा में कई विधीय(methords) हैं जो आपको संख्याओं पर गणितीय कार्य करने की अनुमति देते हैं

  Math.max(x,y)| गणित.ज़्यादा(x,y):

This method is used to find the biggest value out of x and y: इस विधि का उपयोग x और y के सबसे बड़े को खोजने के लिए किया जाता है:।

Example:

उदाहरण:
गणित.ज़्यादा(x,y);

  Math.min(x,y)| गणित.कम(x,y):

This method is used to find the smallest value out of x and y: इस विधि का उपयोग x और y में से सबसे छोटा मान खोजने के लिए किया जाता है:

Example:

उदाहरण:
गणित.कम(x,y);

  Math.sqrt(x)| गणित.मूल(x)(वर्गमूल):

This method is used to find the square root of x: इस विधि का उपयोग x वर्गमूल को खोजने के लिए किया जाता है:

Example:

उदाहरण:
गणित.मूल(x,y);

  Math.abs(x)| गणित.शुद्ध(x)(परिशुद्ध):

This method is used to find the absolute (positive) value of x: इस विधि का उपयोग x के निरपेक्ष (धनात्मक) को खोजने के लिए किया जाता है:

Example:

उदाहरण:
गणित.शुद्ध(x,y);

  Math.random(x)| गणित.कोई(x)(कोई भी अंक):

This method is used to get a random number between 0.0 (inclusive), and 1.0 (exclusive): इस विधि का उपयोग 0.0 (समावेशी), और 1.0 (अनन्य) के बीच के किसी भी अंक को लाने के लिए किया जाता है :

Example:

उदाहरण:
गणित.कोई(x);

So as to get more control over the random number, let us assume , if we only want a random number between 0 and 100, we can use the formula below: उस संख्या पर अधिक नियंत्रण प्राप्त करने के लिए, मान लें, यदि हम केवल 0 और 100 के बीच का कोई अंक चाहते हैं, तो हम नीचे दिए गए उधारण का उपयोग कर सकते हैं:

Example:

उदाहरण:
इंट randomNum = (इंट)(गणित.कोई() * 101); // 0
