 Classes/Objects|कक्षाएं / वस्तुएं:

Hindi4all is an object-oriented(वस्तु-आधारित) programming language.
Everything in here is associated with कक्षा/वर्ग(classes) and objects(उनकी वस्तुएं), along with its attributes and methods. For example: in real life, a car is an वस्तु(object). The car has attributes, such as weight and color, and methods(विधियाँ) or we can say functions, such as drive and brake.
A Class(कक्षा/वर्ग) is like an object constructor, or a "blueprint" for creating वस्तु(objects). Hindi4all एक वस्तु-आधारित(object-oriented) प्रोग्रामिंग भाषा है।
यहाँ सब कुछ कक्षा/वर्ग(class) , उनकी वस्तुए(objects) और उसके साथ-साथ उन वस्तुओ की विशेषता और विधियों से जुड़ा है। उदाहरण के लिए: वास्तविक जीवन में, कार एक वस्तु (object) है। कार में विशेषताएँ(या कहे गुण) होते हैं, जैसे वजन, रंग, और विधियाँ(methords) जैसे चलाना(drive) और रोकना(ब्रेक) आदी । एक कक्षा/वर्ग(class) एक वस्तु निर्माणकर्ता की तरह है, या कहे कोई वस्तु(object) बनाने के लिए एक "रूपरेखा" प्रदान करता है।

Example:

Create a वर्ग/कक्षा(class) called "मुख्य" with variable: अ

उदाहरण:

"मुख्य" नामक एक वर्ग/कक्षा(class) बनाएँ:अ and आ
कक्षा मुख्य { अंक अ =10; }

Note :

कक्षा is same as वर्ग we can any time write कक्षा in place of वर्ग and vice-verasa.

ध्यान दे:

वर्ग , कक्षा के समान है हम किसी भी समय,कही भी वर्ग के स्थान पर कक्षा और कक्षा के स्थान पर वर्ग लिख सकते हैं ।



 Create an Object|एक वस्तु बनाएँ

In Hindi4all,a वस्तु(object) is created from a वर्ग(class). We have already created the वर्ग(class) named "मुख्य", so now we can use this to create वस्तु(object)'s.
To create an वस्तु(object) of "मुख्य", specify the वर्ग(class) name, followed by the वस्तु(object) name, and use the keyword नई:(after this you are also supposed to add (); to it which we would learn later) Hindi4all में एक वर्ग(class) से एक वस्तु(object) बनाई जाती है। हमने "मुख्य" नाम का वर्ग(class) पहले ही बना लिया है, इसलिए अब हम इसका उपयोग वस्तु(object) बनाने के लिए कर सकते हैं।
"मुख्य" का वस्तु(object) बनाने के लिए,वस्तु के वर्ग(class) का नाम निर्दिष्ट करें उसके बाद , वस्तु(object) का नाम , और नया संकेत शब्द(keyword) का उपयोग करें:(इसके बाद आपको इसमें (); भी जोड़ना है जो हम बाद में जानेंगे)

Example:

Create an object called "मेरीवस्तु" and print the value of अ

उदाहरण:

"मेरीवस्तु" नामक एक वस्तु बनाएँ और अ का मान प्रिंट करें
कक्षा मुख्य { अंक अ =10; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु =नई मुख्य(); तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }// 10: प्रदर्शित करेगा



 Multiple Objects|एकाधिक वस्तुएँ

We can create multiple objects of one class: आप एक वर्ग/कक्षा के कई वस्तु(object) बना सकते हैं:

Example:

Create two वस्तु(object)'s of "मुख्य"

उदाहरण:

"मुख्य" की दो वस्तुएँ(objects) बनाएँ
कक्षा मुख्य { अंक अ =10; सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ मुख्य मेरीवस्तु1 = नई मुख्य(); मुख्य मेरीवस्तु2 = नई मुख्य(); तंत्र.बाहर.प्रिंट(मेरीवस्तु1.अ); तंत्र.बाहर.प्रिंट(मेरीवस्तु2.अ); } }// 1010 :प्रदर्शित करेगा // दो दस के बीच जगह के लिए " " का प्रयोग करे //तंत्र.बाहर.प्रिंट(" "+मेरीवस्तु2.अ);



 Using Multiple Classes|एकाधिक वर्गों का उपयोग

We can also create an वस्तु(object) of a वर्ग(class) and access it in another वर्ग(class). This is often used for better organization of classes (one वर्ग(class) has all the attributes(we name it helper/सहायक class) and methods, while the other holds the जरूरी() method/विधि ,i.e. code to be executed). हम वर्ग(class) का वस्तु (वस्तु) भी बना सकते हैं और इसे दूसरे वर्ग(class) में एक्सेस कर सकते हैं। यह अक्सर कक्षाओं के बेहतर संगठन के लिए उपयोग किया जाता है (एक वर्ग(class) में सभी विशेषताएँ/गुण या वेरीअबल हो सकते हैं (हम इसे "सहायक" वर्ग का नाम देंगे) , और दूसरे मे जरूरी() विधि/methord को रख सकते है(हम इसे "मुख्य" वर्ग का नाम देंगे ))।

Example:

Create two वस्तु(object)'s of "मुख्य"

उदाहरण:

"मुख्य" की दो वस्तुएँ(objects) बनाएँ
वर्ग सहायक{ अंक अ =10; } वर्ग मुख्य { सार्वजनिक स्थिर रिक्त जरूरी(डोर[] क ){ सहायक मेरीवस्तु = नई सहायक(); तंत्र.बाहर.प्रिंट(मेरीवस्तु.अ); } }//10 : प्रदर्शित करेगा

Note :

कक्षा is same as वर्ग we can any time write कक्षा in place of वर्ग and vice-verasa.
